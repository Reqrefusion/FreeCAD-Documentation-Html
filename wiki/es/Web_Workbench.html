Indice
The Web Workbench is no longer included after version 0.21.

Introducción

El Ambiente de trabajo Web no es realmente un ambiente de trabajo de modelado, sino que proporciona una ventana del navegador web dentro de FreeCAD.

Desde este navegador eres capaz de navegar por la web, hacer clic en los enlaces, leer la documentación en línea, y entrar en los sitios web (las cookies son compatibles). El ambiente de trabajo Web es también una base para desarrollar herramientas relacionadas con la web en FreeCAD. La ventana del navegador está basada en WebKit, y proporciona botones de navegación estándar como parar, recargar, página anterior y siguiente, y acercar y alejar.

Tools

The Web Workbench offers the following tools:

  • Set URL: Changes the URL of the active Web tab.
  • Open website...: Opens the FreeCAD website.
  • Previous page: Goes to the previous web page.
  • Next page: Goes to the next web page.
  • Refresh web page: Updates the current web page.
  • Stop loading: Stops loading the current web page.
  • Zoom in: Zooms in on the current web page.
  • Zoom out: Zooms out on the current web page.
