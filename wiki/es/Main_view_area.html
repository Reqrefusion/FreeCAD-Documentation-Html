Introducción

El área de visualización principal es el widget principal que contiene subventanas en la interfaz. Es una instancia de Qt QMdiArea.

Normalmente contiene pestañas para la página de inicio de el entorno de trabajo Inicio, y la vista 3D para cada documento abierto. Puede contener otras pestañas de ventanas, por ejemplo, una para cada página de dibujo de TechDraw, hoja de cálculo, o documento de texto.

El área de vista principal se indica con el número 1 y puede contener varias pestañas de ventana, principalmente la página de inicio y la vista 3D.

Ventanas

Las ventanas en el área de vista principal se pueden organizar con comandos ubicados en el menú de ventanas, como Std VentanasMosaico y Std VentanasCascada.
