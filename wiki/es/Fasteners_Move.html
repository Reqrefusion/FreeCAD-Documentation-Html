Descripción

El comando Mover elementos de sujeción mueve y fija un elemento de sujeción a un borde circular. También se puede utilizar para soltar un elemento de sujeción. Un elemento de sujeción tiene un Datosbase Object (objeto base) y su DatosPlacement (posición) está vinculado dinámicamente a ese objeto.

Uso

Attach

A la izquierda se seleccionan un elemento de sujeción y un borde circular. A la derecha, el sujetador se ha movido y fijado al borde seleccionado.

  1. Seleccione un solo elemento de sujeción y un solo borde circular.
  2. Hay varias formas de invocar el comando:
    • Presione el botón Mover elemento de sujeción.
    • Seleccione la opción Fasteners → Mover elemento de sujeción del menú.
  3. El elemento de sujeción se mueve hasta el borde seleccionado y se fija a él.

Detach

  1. Seleccione un solo elemento de sujeción.
  2. Hay varias formas de invocar el comando:
    • Presione el botón Mover elemento de sujeción.
    • Seleccione la opción Fasteners → Mover elemento de sujeción del menú.
  3. El elemento de sujeción está suelto.
