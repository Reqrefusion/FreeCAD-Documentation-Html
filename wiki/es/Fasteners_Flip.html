Descripción

El comando Fasteners Invertir invierte la orientación de los elementos de sujeción cambiando su propiedad Datosinvert.

Uso

  1. Seleccione uno o más elementos de sujeción. La selección puede incluir elementos de sujeción sueltos, pero estos no se voltearán. Ver Notas.
  2. Hay varias formas de invocar el comando:
    • Presione el botón Invertir elemento de sujeción.
    • Seleccione la opción Fasteners → Invertir elemento de sujeción del menú.
  3. Se invierte la orientación de los elementos de sujeción seleccionados.

Notas

  • La propiedad Datosinvert se ignora para los elementos de sujeción no conectados y no se pueden voltear con este comando. Para invertirlos, se debe cambiar su DatosPlacement, por ejemplo con el comando Std TransformManip.
