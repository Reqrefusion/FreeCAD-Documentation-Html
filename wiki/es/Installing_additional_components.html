Introducción

Después de instalar FreeCAD para tu sistema operativo (Windows, Linux o Mac), puedes considerar la instalación de uno o más de los siguientes componentes adicionales.

Archivos de ayuda

La documentación fuera de línea no se envía con todos los instaladores, pero está disponible como un paquete separado. Consulte la página Instalación del archivo de ayuda para obtener más información.

Ambientes de trabajo externos

Aparte de los Ambientes de Trabajo por defecto incluidos en FreeCAD, hay una gran colección de útiles Ambientes de trabajo externos hechos por miembros de la comunidad.

Software de terceros

FreeCAD apoya varios paquetes de software de terceros fuera de la caja. En muchos casos todo lo que necesitas hacer es instalar el software, y cuando FreeCAD se reinicie lo encontrará automáticamente y será capaz de utilizarlo. Esta sección tiene como objetivo proporcionar una lista de tales paquetes de software, junto con alguna información sobre dónde se utilizan en FreeCAD y dónde se pueden descargar.

Apoyo

GitPython

GitPython es una biblioteca para interactuar con los repositorios Git. El Gestor Complementos puede utilizar esta biblioteca. GitPython está incluido en los instaladores de FreeCAD para Windows y Mac.

GraphViz

GraphViz es un software de visualización gráficos de código abierto. Es utilizado por la herramienta Std Gráfico dependencia.

OpenCAMLib

OpenCAMLib es una biblioteca de código abierto de algoritmos de fabricación asistida por ordenador (CAM). Se utiliza en el Ambiente de Trabajo Path. Consulte la página OpenCamLib para obtener instrucciones de instalación.

OpenSCAD

OpenSCAD es un modelador de sólidos en 3D. El OpenSCAD Ambiente de trabajo depende de este software y el Ambiente de trabajo Malla lo utiliza para sus herramientas booleanas. También es necesario para la importación de archivos SCAD con la herramienta Std Importar.

Formatos de archivo

Todo el software de esta sección será utilizado por las herramientas Std Importar o Std Exportar.

CADExchanger

CADExchanger es una aplicación comercial para intercambiar varios formatos de archivos CAD. Existe un ambiente de trabajo externo para utilizar esta aplicación en FreeCAD.

DXF Importador

FreeCAD tiene un importador y exportador nativo para archivos DXF, programado en C++. Actualmente no implementan todas las características del formato DXF. Para esas características el importador y el exportador heredados de Python están todavía disponibles. Estos requieren la biblioteca Python Draft-dxf-importer. Ver la página FreeCAD y DXF Import para más información.

DWG converters

FreeCAD cannot directly read and write DWG files. To convert DXF files to DWG files, and vice-versa, FreeCAD relies on external converters. There is built-in support for the following DWG converters:

  • LibreDWG (open-source, lacks support for some DWG entities).
  • ODA File Converter (free, but not open-source).
  • QCAD pro (commercial). introduced in version 0.20

See Import Export Preferences and FreeCAD and DWG Import for more information.

IfcOpenShell

IfcOpenShell es una biblioteca para trabajar con el formato de archivo Industry Foundation Classes (IFC) utilizado en el diseño arquitectónico. La biblioteca también es utilizada por el Arch IfcExplorer (version 0.18 y abajo) y BIM IfcExplorer. IfcOpenShell está incluido en los instaladores de FreeCAD para Windows y Mac.

IfcJson

IfcJson es una biblioteca necesaria para exportar al formato de archivo IFCJSON. IFCJSON es un nuevo formato IFC que aún no es soportado por muchas aplicaciones.

Pycollada

Pycollada, también conocido como python-collada, es una biblioteca de Python para leer y escribir archivos Collada (DAE). Pycollada se incluye en los instaladores de FreeCAD para Windows y Mac.

Renderización

LuxCoreRender

LuxCoreRender

LuxCoreRender es una máquina de renderizado, reinicio del proyecto LuxRender. Oficialmente no está soportado por el Ambiente de trabajo Trazado de rayos, pero podría valer la pena probarlo. Está oficialmente soportado por el nuevo Ambiente de trabajo Renderización, pensado como un futuro reemplazo del ambiente de trabajo trazado de rayos. Ver la página LuxCoreRender para más información e instrucciones de instalación.

LuxRender

LuxRender

LuxRender es una de las dos máquinas de renderización soportados por el Ambiente de trabajo Trazado de rayos. En 2013 el proyecto ha sido reiniciado convirtiéndose en LuxCoreRender, con una importante reescritura de código y cambios que rompen la compatibilidad. Oficialmente el ambiente de trabajo trazado de rayos sólo soporta el abandonado LuxRender (la última versión es la 1.6, 2017-12-28), mientras que el nuevo Ambiente de trabajo Renderización (pensado como un futuro reemplazo del Ambiente de trabajo Trazado de rayos) soporta en cambio LuxCoreRender y ha dejado de soportar LuxRender. De todos modos, aunque oficialmente no esté soportado, LuxCoreRender puede funcionar con el Ambiente de trabajo Trazado de rayos, podría valer la pena probarlo. Mira la página de LuxRender para más información e instrucciones de instalación, y la de LuxCoreRender si quieres probar un software más moderno.

POVRay

POV-Ray es un conocido trazador de rayos que puede renderizar imágenes fotorrealistas. Es una de las dos máquinas de renderización actualmente soportados por el Ambiente de trabajo Trazado de rayos. Vea la página POV-Ray para más información e instrucciones de instalación.

Elemento Finito

CalculiX

CalculiX es un conjunto de dos paquetes de elementos finitos: CalculiX CrunchiX, un solucionador MEF, y CalculiX GraphiX, una interfaz gráfica de usuario. Sólo el solucionador está soportado por FreeCAD. Es utilizado por la herramienta Solucionador CalculiX.

Gmsh

Gmsh es un generador automático de mallas de elementos finitos. Es utilizado por las herramientas MEF MallaGmshDesdeForma y Malla DesdePiezaForma.

Elmer

Elmer es un software de simulación multifísica, de código abierto en 2005. En FreeCAD sus módulos Rejilla y Solucionador son utilizados por la herramienta FEM SolverElmer.

FEniCS

FEniCS es una plataforma de computación para resolver ecuaciones diferenciales parciales (EDP), que se utilizan ampliamente al resolver problemas de MEF. Es utilizado por el Ambiente de trabajo MEF

Z88

Z88 es otro programa de MEF que contiene un mallado, un solucionador y convertidores. Es utilizado por la herramienta FEM SolucionadorZ88. FreeCAD requiere el paquete Z88OS de código abierto.

OpenFOAM

OpenFOAM es una gran colección de bibliotecas para simulaciones de dinámica de fluidos computacional (CFD). OpenFOAM es utilizado por los Cfd y CfdOF Ambientes de trabajo externo.
