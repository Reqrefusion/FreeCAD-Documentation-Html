Description

The Spreadsheet StyleBold tool toggles the contents of selected cells to/from bold.

Usage

  1. Make sure there is an active spreadsheet.
  2. Select one or more cells.
  3. There are several ways to invoke the tool:
    • Press the Bold text button.
    • Select the Spreadsheet → Styles → Bold text option from the menu.
    • Use the keyboard shortcut: Ctrl+B. introduced in version 1.1
