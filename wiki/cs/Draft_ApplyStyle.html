Description

Popis

Tento nástroj aplikuje aktuálně nastavenou barvu a styl čáry na vybrané objekty.

version 0.21 and below: This command handles only five of the settings the Draft SetStyle command offers.

introduced in version 0.21: This command changes the view properties of objects. It applies all settings the Draft SetStyle command offers. It also changes these additional properties:

  • PohledDecimals (for dimensions): See Draft Preferences.
  • PohledShowLine (for dimensions): Idem.

Usage

Použití

  1. Na aktivní pracovní ploše Kreslení nebo Architektura nastavte požadovanou barvu a styl čáry v panelu Nástrojů (nebo v pruhu nástrojů Kreslení, pokud používáte mód nástrojových pruhů).
  2. Vyberte jeden nebo několik objektů
  3. Stiskněte tlačítko Použij styl

Notes

  • The Draft SetStyle command can also apply settings.
