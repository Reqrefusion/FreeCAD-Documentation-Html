Description

The Part RuledSurface command creates a ruled surface spanning between two selected edges/wires.

Usage

  1. Select two edges or wires (open or closed contours).
  2. There are several ways to invoke the command:
    • Press the Part RuledSurface button
    • Use the Part → RuledSurface entry in the Part menu
  3. A ruled surface is created between the selected edges/wires.

Notes

  • App Link objects linked to the appropriate object types can also be used as source objects.
