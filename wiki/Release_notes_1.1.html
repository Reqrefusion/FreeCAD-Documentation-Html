Draft Workbench

Further Draft improvements

  • Support for relative font paths has been added to ShapeStrings. Pull request #17819
  • The Draft Fillet command now works on selected edges, instead of the first edge of selected objects. Pull request #17945 and Pull request #18150

FEM Workbench

Further FEM improvements

  • Log verbosity can now be set for Gmsh in the Preferences. Pull request #17699
  • The DataSecond Order Linear property and support for local refinement, previously only available for Gmsh, are now also available for the new Netgen implementation. Pull request #17170

Material Workbench

Further Material improvements

  • Physical properties based on the material are now displayed in the property view. This includes the material name, density, mass, and volume. The default density is 0. Pull request #17438

Sketcher Workbench

placeholder for image/gif Create defining external geometry: It is now possible to create 'real' external geometry in Sketcher and toggle between defining and construction mode.

Pull request #17736

placeholder for image/gif Create intersection external geometry: It is now possible to create external geometry in Sketcher which is based on the selected reference geometry and the intersection of the sketch plane. The default mode is the current one: create a projection of the external feature onto the sketch plane.

Pull request #17736

placeholder for image/gif Create external geometry from face selection: It is now possible to create external geometry (both projection and intersection) in Sketcher by selecting a face.

Pull request #17736

Further Sketcher improvements

  • It is now possible to directly use external geometry as input for tools like offset or transform (array), for both external construction and defining geometry Pull request #17615
  • The Sketcher axes are now displayed with infinite length. Pull request #17312
