Introduction

介绍

这是FreeCAD 的文档wiki。有几种方法可以使用这个文档：通过探索枢纽，通过遵循手册，或者通过使用FreeCAD内部的帮助菜单条目。这是一项正在进行中的工作，由FreeCAD的用户和开发人员组成的社区编写。如果你发现有错误或遗漏的信息，请帮助 FreeCAD!

核心

使用者入口：本页包含对FreeCAD用户有用的一般文档：所有工作台的列表、关于如何安装和使用FreeCAD应用程序的详细说明、教程以及所有你需要开始的东西。

发烧友入口：这个页面收集了高级用户和对编写Python脚本感兴趣的人的文档。在那里你还可以找到一个宏库，关于如何安装和使用它们的说明，以及关于根据你的具体需要定制FreeCAD的更多信息。

开发者入口：本节包含为开发者提供的材料： 如何自己编译FreeCAD，FreeCAD源代码的结构，如何浏览源代码，如何开发新的工作台，以及将FreeCAD嵌入自己的应用程序。

书籍手册

FreeCAD 手册 是另一种更线性的方式，来呈现此维基中的信息。它做成了一本书的形式，也会善意地引导您进入上面三个入口中的许多其他页面。 这里有 电子书版本，还有 几个翻译后的PDF格式。

参与进来

怎样参与

如果您有兴趣帮助我们，那在 FreeCAD 项目中可以做很多事情。这里当然有给 C++ 或 Python 程序员的编程任务，但是即使你不能写代码，这里也有许多事情你可以做，比如：

  • 撰写文档和 编辑 维基(en)；
  • 帮助新来的人；
  • 翻译应用程序和文档；
  • 为您最喜欢的操作系统，帮助打包最新版本的 FreeCAD；
  • 帮助您周围的人发现 FreeCAD。

帮助 FreeCAD 页面描述了所有更多的细节。从 2016 年开始，FreeCAD 参与 谷歌编程夏令营。

源代码

FreeCAD 可以用 CMake 在所有平台上编译。源代码是 LGPL 授权的，并在 GitHub 上托管，并且有镜像在： GitLab 和 CodeBerg 。有 Windows，Linux 和 MacOS 的构建指导。源代码文档托管在 这里，由 Doxygen 生成，记录在 这个 维基页面上。

关于开发

通过 开发路线图，看看什么正在计划之中。通过 修改记录、路线图页 和 FreeCAD跟踪器 页面，查看下一个版本的进展。或者，通过 项目统计 寻找有关 FreeCAD 代码库的更多信息。所有的开发交流都在 论坛 上，如果您有兴趣参与，请务必访问那里。
