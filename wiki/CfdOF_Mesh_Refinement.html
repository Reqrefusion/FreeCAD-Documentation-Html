Description

The Mesh refinement dialogue panel is used to add surface, volume and extrusion refinements to the background mesh.

Usage

The dialogue panel offers the following settings:

Once the CFD Mesh has been created, the Mesh refinement dialogue panels can be added to the tree in the following ways:

  1. Select the [Geometry Name]_Mesh within the tree
  2. Both the button and the CfdOF → Mesh refinement‏‎ menu option, become available.
  3. Click on either the:
    • button or,
    • CfdOF → Mesh refinement‏‎ menu option.
  4. The Mesh refinement dialogue panel will open.
  5. MeshRefinement will be created in the tree below the [Geometry Name]_Mesh.
  6. Multiple Mesh refinements can be added to the tree. Each additional Mesh refinement will be named MeshRefinementxxx where xxx is an incrementing number from 001.
