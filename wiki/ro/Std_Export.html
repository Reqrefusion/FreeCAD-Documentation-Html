Descriere

Aceasta este comanda standard pentru a exporta obiecte din interiorul unui document FreeCAD în alte formate de fișiere. O selecție trebuie făcută înainte de lansarea comenzii.

Cum se folosește

  1. Selectați obiectele pentru a fi exportate, fie în arborescența Model sau în vizualizarea 3D utilizând box selection.
  2. Mergeți în meniu pe traseul File → Export
  3. Selectați tipul de fișier
  4. Tastați un nume
  5. Apăsați butonul Save.

Options

  • Press Esc or the Cancel button to abort the command.

Notes

  • To export a mesh object to a solid file format it must first be converted. See the Import from STL or OBJ tutorial.
  • Some workbenches have additional export commands. See Import Export.

Preferences

  • See Import Export Preferences.
