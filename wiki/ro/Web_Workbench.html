The Web Workbench is no longer included after version 0.21.

Introduction

Panoul Internet furnizează o fereastră de navigare pe reţeaua globală prin intermediul FreeCAD, bazat pe WebKit. 

Pornind de aici, puteţi parcurge întregul Internet, să vă autentificaţi pe situri (acceptă "cookies"). Panoul Internet reprezintă totodată o bază pentru dezvoltarea uneltelor legate de web în FreeCAD. Navigatorul pe Internet vă oferă o serie de butoane obişnuite, precum "Stop", "Reîncarcă", "Pagina precedentă" sau "Pagina următoare".

Tools

The Web Workbench offers the following tools:

  • Set URL: Changes the URL of the active Web tab.
  • Open website...: Opens the FreeCAD website.
  • Previous page: Goes to the previous web page.
  • Next page: Goes to the next web page.
  • Refresh web page: Updates the current web page.
  • Stop loading: Stops loading the current web page.
  • Zoom in: Zooms in on the current web page.
  • Zoom out: Zooms out on the current web page.
