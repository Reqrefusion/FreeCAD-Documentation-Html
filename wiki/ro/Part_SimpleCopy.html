Introducere

Elimină istoricul piesei, astfel încât pașii diferiți pentru a crea piesa nu mai sunt accesibili / editabili de loc.

The Part SimpleCopy command creates non-parametric copies of objects.

Utilizare

Mark part of choice in the tree view and choose Part → Create simple copy.

  1. Select one or more objects.
  2. Select the Part → Create a copy → Create simple copy option from the menu.

Notes

  • For objects nested in containers Part TransformedCopy should be used instead.

Properties

See also: Property editor.

The created objects are Part Feature objects with no additional properties.
