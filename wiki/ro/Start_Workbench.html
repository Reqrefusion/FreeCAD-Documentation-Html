The Start Workbench is no longer included after version 0.21.
It has been replaced by the Start page that can be accessed via the Help menu.

Start Workbench nu este un atelier propriu zis, este doar pagina care este afișată atunci când deschideți FreeCAD fără a avea un document încărcat.

Din această interfață puteți vedea informații utile și puteți să accesați fișierele, exemplele, documentația și cele mai recente știri din lumea FreeCAD, deschise recent. Aveți posibilitatea să modificați modul în care se comportă Workbench Start sau să schimbați atelierul de lucru implicit înPreferences Editor în meniul Edit → Preferences → Start.
