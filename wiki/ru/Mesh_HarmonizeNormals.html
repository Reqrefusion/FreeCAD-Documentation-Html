Description

The Mesh HarmonizeNormals command harmonizes the normals of mesh objects.

Usage

  1. Select one or more mesh objects.
  2. There are several ways to invoke the command:
    • Press the Harmonize normals button.
    • Select the Meshes → Harmonize normals option from the menu.

Notes

  • This command can produce a mesh with flipped normals. The Mesh FlipNormals command can be used to correct this.
  • For a clear indication of the orientation of the faces of mesh objects make sure the ВидLighting property of the mesh objects is set to One side. The color of the back side of their faces will then depend on the backlight settings: Edit → Preferences... → Display → 3D View → Backlight color - Intensity. See: Preferences Editor.
