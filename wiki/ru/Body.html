Введение

Во FreeCAD слово "Тело" обычно используется для ссылки на объект Тело PartDesign (класс PartDesign::Body), определённый в Верстаке PartDesign. Это контейнер, который может содержать 2D эскизы и 3D геометрические операции для создания твердотельной формы.

Смотри Тело PartDesign для получения дополнительной информации об этом типе объекта.

Применение

  1. Переключитесь на верстак PartDesign.
  2. Нажмите Создать тело.
  3. Нажмите Создать эскиз для создания нового эскиза.
  4. Создайте замкнутую полилинию, потом используйте Выдавливание для выдавливания эскиза и создания базового твёрдого тела.
  5. Добавьте ещё эскизы и выдавливания, и используйте другие инструменты верстака PartDesign для модификации и трансформации исходного тела.

В качестве альтернативы, вместо использования эскизов, можно добавить конструктивный элемент PartDesign, например, Аддитивный куб. Для создания конечной формы можно использовать любое количество аддитивных и субтрактивных конструктивных элементов.

Примечания

Тело требуется при использовании Верстака PartDesign с методами редактирования конструктивных элементов.

Тело не требуется при использовании верстака Part, поскольку этот верстак использует процедуру конструктивной блочной геометрии, базирующуюся на примитивах и булевых операциях.
