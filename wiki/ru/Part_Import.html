Описание

Этот инструмент позволяет Вам добавить в текущий документ файлы *.IGES, (стандарт) *.STEP, *.BREP.

Применение

  1. Выберите в верхнем меню Деталь → Импорт CAD.
  2. Выберите файл для импорта

Notes

  • The Std Import command supports many more file formats.
