Каждая операционная система имеет свою собственную страницу по компиляции FreeCAD в Онлайновой документации.

Compile on Windows Compile on Linux Compile on MacOS Compile on Docker

Чтобы скомпилировать в BSD или другой системе Unix, Вы можете следовать общим инструкциям из Compile on Linux, но используя соответствующие зависимости в этих системах.

Так же возможна компиляция FreeCAD в Windows с использованием Compile on Windows.

Чтобы скомпилировать FreeCAD без применения графического интерфейса, используйте Docker образ FreeCAD в режиме командной строки.

Насчёт компиляции программной документации смотрите Source documentation.

Чтобы научиться использовать систему контроля версий и предлагать код, смотрите Поддержка исходного кода.
