Описание

Переместить объект за другой объект, при выделении этой команды в контекстном меню, позволяет переупорядочивать объекты в Теле. Перемещать можно эскизы, геометрию построения и конструктивные элементы.

Применение

  1. In the Model tree, select the object(s) to be moved then right-click to open the context menu.
  2. Select Move object after other object.
  3. In the Select feature dialog, select the object under which to move the object(s).
  4. Press OK.
