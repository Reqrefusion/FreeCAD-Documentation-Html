Описание

Инструмент «Библиотека BIM» позволяет разместить в текущей модели объект из FreeCAD Parts Library, который должен быть установлен с помощью Addon Manager, чтобы этот инструмент был доступен.


Выше: Смотрите диалог Library browser панели задач слева

Применение

  1. Нажмите кнопку BIM Library
    Result: In the Combo view → Task panel a dialog titled Library browser will display
  2. Щелкните файл в браузере библиотеки
  3. Дважды щелкните его или нажмите кнопку Insert
  4. Щелкните точку в трехмерном виде или введите координату вручную, чтобы поместить объект

Опции

  • Поддерживаются файлы FCStd, STEP и BREP. Доступны для размещения только файлы STEP и BREP. Файлы FCStd не позволят вам выбрать место размещения, так как они могут состоять из сложной серии объектов со значительными позициями. При выборе файла FCStd его содержимое будет вставлено в позицию, определенную в файле.
  • Объекты STEP и BREP вставляются как Equipment, без отдельной формы основания. Добавление базовой формы к этим объектам впоследствии уничтожит их текущую форму.
  • При размещении объекта вы можете выбрать их точку вставки между оригинальной (точкой (0,0,0), определенной в файле), верхней, средней, нижней и левой, центральной и правой.
  • Библиотека может работать в автономном режиме, в этом случае она зависит от устанавливаемой надстройки библиотеки компонентов (и обновляется пользователем), или в сети, и в этом случае она просматривается непосредственно с репозитория Git библиотеки Parts и позволяет скачивать прямо оттуда.
