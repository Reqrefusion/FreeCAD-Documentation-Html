Template:Touchpad Navigation

{{{Select_name}}} {{{Zoom_name}}} {{{Rotate_view_name}}} {{{Pan_name}}}
{{{Ctrl}}}+{{{Shift}}}+ {{{Alt}}}+ {{{Shift}}}+ {{{Shift}}}+
{{{Select_text}}} {{{Zoom_text}}} {{{Rotate_view_text}}} {{{Pan_text}}}

This template creates the table that shows the Touchpad navigation style.

The text has to be given in the specific fields. In this way the template can be translated into different languages.

Example: 

{{Touchpad Navigation
|Select_name=Select
|Zoom_name=Zoom
|Rotate_view_name=Rotate view
|Pan_name=Pan

|Ctrl={{KEY|Ctrl}}
|Shift={{KEY|Shift}}
|Alt={{KEY|Alt}}

|Select_text=Press the left mouse button over an object you want to select.

|Zoom_text=Hold {{KEY|Ctrl}} and {{KEY|Shift}}, then move the pointer.

|Rotate_view_text=Hold {{KEY|Alt}}, then move the pointer.

Alternatively, hold {{KEY|Shift}} and the left button, then move the pointer.

|Pan_text=Hold {{KEY|Shift}}, then move the pointer.
}}

Result:

Select Zoom Rotate view Pan
Ctrl+Shift+ Alt+ Shift+ Shift+
Press the left mouse button over an object you want to select. Hold Ctrl and Shift, then move the pointer. Hold Alt, then move the pointer.

Alternatively, hold Shift and the left button, then move the pointer.

Hold Shift, then move the pointer.
