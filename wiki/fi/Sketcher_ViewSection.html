Kuvaus

View section piilottaa luonnostason edessä olevan näkyvistä.

Vasemmalla luonnos näkyy vain osittain ja oikealla View section:in avulla kokonaan.

Usage

See also: Sketcher Preferences.

  1. This tool is only available if a sketch is in edit mode.
  2. There are several ways to invoke the tool:
    • Press the View section button.
    • Select the Sketch → View section option from the menu.
