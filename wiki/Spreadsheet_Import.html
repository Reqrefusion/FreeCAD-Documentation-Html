Description

The Spreadsheet Import tool imports a CSV file into a new spreadsheet. Note that the tab character, and not the comma, is the default field delimiter. But this can be changed in the Spreadsheet Preferences (introduced in version 0.20).

Usage

  1. There are several ways to invoke the tool:
    • Press the Import spreadsheet button.
    • Select the Spreadsheet → Import spreadsheet option from the menu.
  2. Select a CSV file.
  3. Press the Open button.

Notes

  • Use the Std Import command to import Excel *.xlsx files.
