FAQ

How do I change the direction the door opens?

v1.0

Two ways: 

Note: Before following these steps, it may be helpful to set the following values to better visualize the window/door opening direction while making these changes: DataSymbol Plan=True, DataOpening=45 (or another value greater than 0)
  • Edit Mode method
  1. Double-click the feature in the tree view to enter Edit Mode.
  2. click the button "Invert opening direction" in the Edit Mode window.
  • Tree method
  1. Right-click the feature in the tree view.
  2. choose the "Invert opening direction" from the context menu to change the direction.
