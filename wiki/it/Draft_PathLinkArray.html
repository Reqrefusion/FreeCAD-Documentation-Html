Descrizione

Il comando Serie di link su tracciato crea un array Link da un oggetto selezionato posizionando le copie lungo un percorso. Utilizzare invece il comando Copie su tracciato per creare un array regolare meno efficiente. Ad eccezione del tipo di array creato, Link array o array regolare, questo comando è identico al comando Copie su tracciato. Vedi lì per ulteriori informazioni.
