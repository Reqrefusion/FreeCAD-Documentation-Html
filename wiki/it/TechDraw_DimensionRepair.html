Descrizione

Lo strumento TechDraw Ripara la quota può ripristinare i riferimenti geometrici 2D o 3D di una quota. Questi riferimenti potrebbero diventare errati a causa del "problema di denominazione topologica" o della rimozione delle linee nascoste.

Vedere TechDraw Quota allineata per ulteriori informazioni sulla quotatura e sulla denominazione topologica.

Utilizzo

  1. Selezionare la quota da correggere e, facoltativamente, i nuovi riferimenti geometrici, punti o bordi, nella Vista TechDraw o nella vista 3D.
  2. Esistono diversi modi per richiamare lo strumento:
    • Premere il pulsante Ripara riferimenti quota.
    • Selezionare l'opzione TechDraw → Quotatura → Ripara riferimenti quota dal menu.
  3. Si apre il pannello delle attività Riparazione quota che mostra la quota selezionata e i riferimenti alla geometria corrente.
  4. Selezionare nuovi riferimenti geometrici se non lo si ha già fatto.
  5. Premere il pulsante Sostituisci i riferimenti con la selezione corrente.
  6. Premere il pulsante OK per aggiornare la quota.
  7. Se sono stati selezionati riferimenti 3D: facoltativamente modificare la proprietà DatiMeasure Type della quota in True.

Script

Vedere anche: Autogenerated API documentation e Script di base per FreeCAD.

Lo strumento TechDraw Ripara la quota non può essere utilizzato in macro o dalla console Python. Le proprietà di riferimento delle quote possono essere aggiornate utilizzando Python.
