Descrizione

Asymptote è un linguaggio vettoriale per la computer grafica 2D e 3D. Il codice Asymptote può essere incluso nei documenti LaTeX o utilizzato per generare PostScript, PDF , SVG, WebGL e PRC. I file PDF 3D interattivi creati dal codice Asymptote richiedono Acrobat Reader versione 9 o successiva.

Il supporto per Asymptote è stato aggiunto nella versione 0.19 di FreeCAD.

Esportazione

  1. Facoltativamente, assegnare i colori alle facce dell'oggetto che si desidera esportare con il comando Part: Colore per faccia.
  2. Passare a Mesh.
  3. Creare una mesh dall'oggetto con il comando Mesh: Crea mesh da una forma.
  4. Selezionare il nuovo oggetto mesh.
  5. Richiamare il comando Mesh: Esporta mesh.
  6. Selezionare il formato file *.asy nella finestra di dialogo.
  7. Inserire un nome di file.
  8. Premere il pulsante Salva.

Conversione

È necessario il compilatore Asymptote per convertire i file *.asy. Per convertire in PDF è necessario anche un sistema [1].

Il compilatore è uno strumento da riga di comando. Per convertire in PDF puoi usare questa sintassi: 

PathToAsyExecutable/asy -f pdf AsymptoteFileName.asy
