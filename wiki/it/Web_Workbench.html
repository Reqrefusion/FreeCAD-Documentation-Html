Introduzione

L' Ambiente Web non è un vero ambiente di lavoro, ma fornisce una finestra del browser all'interno di FreeCAD.

Da questo browser è possibile navigare sul Web, fare clic sui collegamenti, leggere la documentazione online e accedere ai siti Web (sono supportati i cookie). L'Ambiente Web è anche una base per lo sviluppo di strumenti relativi al Web in FreeCAD. La finestra del browser è basata su QtWebEngine e fornisce pulsanti di navigazione standard come stop, ricarica, pagina precedente e successiva e zoom avanti e indietro.

Strumenti

L'Ambiente Web offre i seguenti strumenti:

  • Imposta URL: cambia l'URL della scheda Web attiva.
  • Apri sito Web...: Apre il sito Web FreeCAD.
  • Pagina precedente: va alla pagina web precedente.
  • Pagina successiva: va alla pagina Web successiva.
  • Aggiorna pagina web: aggiorna la pagina web corrente.
  • Interrompi caricamento: interrompe il caricamento della pagina Web corrente.
  • Zoom in: ingrandisce la pagina Web corrente.
  • Zoom indietro: Rimpicciolisce la pagina Web corrente.
