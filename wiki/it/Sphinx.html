Introduzione

Sphinx è un sistema di documentazione simile a Doxygen, progettato principalmente per documentare il codice Python. Con l'uso di "Breathe" può anche essere utilizzato per analizzare il codice C++, quindi può essere utilizzato in progetti che mescolano C++ e Python.

Visitare il sito web Sphinx per saperne di più sul sistema e consultare il Manuale Sphinx per tutte le informazioni.

Sphinx e FreeCAD

Questo documento fornisce una breve introduzione a Sphinx, in particolare al modo in cui viene utilizzato in FreeCAD per documentare le sue fonti. Visitare la pagina Documentazione del codice sorgente per istruzioni sulla creazione della documentazione di FreeCAD, che è anche ospitata online sul sito web API di FreeCAD.
