Descrizione

Lo strumento TechDraw Immagine inserisce un'immagine bitmap (PNG, TIFF, JPEG, ecc.) da un file nella pagina come una vista

disponibile dalla versione 1.0: Anche lo strumento TechDraw Vista può anche creare una Vista Immagine.

Immagine inserita nella pagina di disegno

Uso

  1. Se nel documento sono presenti più pagine di disegno: facoltativamente attivare la pagina desiderata selezionandola nella Vista ad albero.
  2. Selezionare l'opzione TechDraw → Viste TechDraw → Inserisci Immagine Bitmap dal menu.
  3. Se nel documento sono presenti più pagine di disegno e non si ha ancora attivato una pagina, si apre la finestra di dialogo Scelta pagina:
    1. Selezionare la pagina desiderata.
    2. Premere il pulsante OK.
  4. Si apre un browser di file.
  5. Selezionare un file immagine.
  6. Viene inserita una Vista Immagine.
  7. Facoltativamente, modificare la sua proprietà DatiScale per regolarne le dimensioni.

Proprietà

Vedere anche: Editor delle proprietà.

Una Vista Immagine, formalmente un oggetto TechDraw::DrawViewImage, ha le proprietà comuni a tutti i tipi di vista. Ha inoltre le seguenti proprietà aggiuntive:

Dati

Image

  • DatiImage File (File): il file contenente questa bitmap.
  • DatiImage Included (FileIncluded): file immagine incorporato. A solo uso del sistema.
  • DatiWidth (Float): la larghezza dell'immagine ritagliata in mm. Utilizzato solo se VistaCrop è true.
  • DatiHeight (Float): l'altezza dell'immagine ritagliata in mm. Idem.

Vista

Image

  • VistaCrop (Bool): ritaglia l'immagine a DatiWidth x DatiHeight.

Script

Vedere anche: Autogenerated API documentation e Script di base per FreeCAD.

Lo strumento Immagine può essere utilizzato nelle macro e dalla console Python tramite la seguente funzione: 

dvi = FreeCAD.ActiveDocument.addObject('TechDraw::DrawViewImage','TestImage')
rc = page.addView(dvi)
dvi.ImageFile = "pathToMy/imageFile.png"
dvi.Height = 200
dvi.Width  = 200
