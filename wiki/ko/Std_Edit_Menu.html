소개

표준 편집 메뉴(Std Edit Menu)는 표준 메뉴의 7개 하위 메뉴 중 하나입니다.

파일     편집     보기     도구     매크로     창     도움말

편집 메뉴는 모델을 변경하는 도구를 제공합니다.

도구

이 메뉴에서는 다음 도구를 사용할 수 있습니다.

  • 실행 취소: 작업 하나를 실행 취소합니다.
  • 다시 실행: 실행 취소한 작업을 다시 실행 합니다.
  • 잘라내기: 스프레드시트 셀을 클립보드로 복사한 후 해당 셀을 지웁니다.
  • 복사: 개체를 클립보드로 복사합니다.
  • 붙여넣기: 클립보드의 개체를 문서로 붙여넣습니다.
  • 개체 복제: 문서의 개체를 복제합니다.
  • 새로 고침: 문서를 다시 계산합니다.
  • 개체 선택 상자: 사각 영역을 그려 개체를 선택합니다.
  • 면 선택 상자: 사각 영역을 그려 면을 선택합니다.
  • 모두 선택: 모든 개체를 선택합니다.
  • 삭제: 선택한 개체를 삭제합니다.
  • 파이썬 콘솔로 보내기: 개체를 참조하는 변수를 파이썬 콘솔에 생성합니다. 0.19 버전부터 도입
  • 위치 설정...: 개체를 위한 위치 설정 작업 패널을 엽니다.
  • Transform: Transforms an object in the 3D view.
  • 정렬...: 고정된 참조 개체를 기준으로 한 개체를 정렬합니다.
  • 편집 모드 전환: 개체의 편집 모드를 활성화 또는 비활성화합니다.
  • Properties: Displays the Property view panel. introduced in version 1.0
  • Edit mode: Defines the edit mode to be used when an object is double-clicked in the Tree view. introduced in version 0.20
  • 환경 설정...: 속성 편집기를 엽니다. -- MacOS에서는 이 메뉴 옵션이 편집 메뉴 대신 FreeCAD 메뉴에 나타날 수 있습니다.
