소개

외부 작업대는 기본 FreeCAD 소스 코드에 통합되지 않은 고급 사용자가 만든 작업대입니다.

이러한 작업대는 핵심 FreeCAD 개발 팀에서 지원되지 않으므로 모든 FreeCAD 버전에서 작동하도록 테스트되지 않았습니다. 질문, 버그 보고, 개선 요청은 워크벤치 작성자에게 직접 해야 합니다.

Workbenches marked with can be installed from the Addon Manager. For manual installation see Installing more workbenches. If there are specific instructions or prerequisites for using an external workbench, then these should be noted on the workbench's home page.

Workbenches marked with are not recommended for recent versions of FreeCAD. They are obsolete, unmaintained, superseded by a newer workbench, or maybe they don't work with Python 3 or with Qt5. In most cases they should be avoided.

조립 작업대

조립 작업대는 일반적으로 서로 호환되지 않습니다. 그 중 하나를 사용하여 조립품을 생성하는 경우 원본 작업대를 고수해야 하며 동일한 문서에서 다른 조립 작업대를 사용해서는 안 됩니다.

부품 간의 관계를 유지하기 위해 실제로 제약 조건을 사용하지 않고 단순히 공간에서 위치를 다시 지정하는 '의사 조립' 작업대도 있습니다.
