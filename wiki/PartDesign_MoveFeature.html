Description

Move object to other body, as this command is labeled in the context menu, allows moving objects under a Body to another Body. The moved object is placed right below the other Body's tip. Sketches, datum geometry and features are supported.

Usage

  1. In the Model tree view, right-click on the object to be moved.
  2. Select Move object to other body from the list in the context menu
  3. In the Select body dialog, select the Body to move the object to from the drop-down menu.
  4. Press OK.
