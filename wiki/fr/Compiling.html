Chaque système d'exploitation dispose de sa page spécifique pour compiler FreeCAD dans la Documentation d'aide en ligne.

Compiler sous Windows Compiler sous Linux Compiler sous Mac Compiler sous Docker

Pour compiler sous BSD et d'autres systèmes Unix, vous pouvez suivre les instructions générales de Compiler sous Linux mais en utilisant les dépendances appropriées dans ces systèmes.

Il est également possible de compiler FreeCAD sous Windows en suivant les instructions de Compiler sous Windows ou en utilisant MSYS2/MinGW après Compiler sous MinGW.

Pour compiler FreeCAD sans son interface utilisateur graphique, utilisez l'image FreeCAD Docker mode CLI.

Pour compiler la documentation de programmation, voir la documentation du code source.

Pour apprendre à utiliser le système de contrôle des révisions et contribuer au code, voir Source code management.
