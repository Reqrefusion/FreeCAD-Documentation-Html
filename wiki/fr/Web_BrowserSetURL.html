Description

Utilisez la commande Web Changer l'URL pour modifier l'URL de l'onglet Web actif.

Utilisation

  1. Assurez-vous qu'un onglet web est actif, utilisez la commande . Web Ouvrir le site web... pour en créer un si nécessaire.
  2. Appuyez sur le bouton Définir l'URL.
  3. Une ligne d'invite s'ouvre avec l'URL en cours.
  4. Modifiez l'URL et appuyez sur Valider pour terminer.
  5. La nouvelle URL est chargée.
