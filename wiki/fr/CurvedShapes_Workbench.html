Introduction

Curved Shapes est un atelier externe de FreeCAD qui crée des formes 3D à partir de courbes 2D.

Installation

L'installation recommandée se fait par le Gestionnaire des extensions de Freecad via Outils → Gestionnaire des extensions

Outils

  • CurvedArray: crée un réseau et redimensionne les éléments dans les limites d'une ou plusieurs courbes de coque.
  • CurvedSegment: interpole entre deux courbes 2D. Les courbes interpolées peuvent être redimensionnées dans les limites de certaines courbes de coque.
  • InterpolatedMiddle: interpole une forme 2D au milieu entre deux courbes 2D. Les formes de base peuvent être connectées à une forme avec un coin pointu.
  • SurfaceCut: coupe une surface pour obtenir la courbe de contour ou une face. Cet outil est similaire à Part Coupes dans l' Atelier Part mais il est entièrement paramétrique et a une option pour réduire la complexité de la courbe de sortie. Il essaie de supprimer les bords qui se chevauchent.

Exemples

  • HortenHIX: script Python qui crée la forme du Horten (également appelé Horten H IX), un chasseur furtif qui a été construit en Allemagne en 1944.
  • FlyingWingS800: script Python qui crée la forme d'un modèle radio commandé d'aile volante.
