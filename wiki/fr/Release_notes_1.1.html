  • Les fonctions de sauvegarde automatique et de sélection additive ont été ajoutées à l'outil Mesurer. Pull request #17717

Atelier Draft

Autres améliorations de Draft

  • La prise en charge des chemins relatifs des polices a été ajoutée à Forme à partir d'un texte. Pull request #17819
  • La commande Draft Congé fonctionne désormais sur les arêtes sélectionnées, au lieu de la première arête des objets sélectionnés. Pull request #17945 et Pull request #18150
  • Le menu des calques de la commande Draft Groupement automatique est trié par ordre alphabétique. Pull request #18172
  • La gestion des liens dans TechDraw Vue d'un objet Draft a été corrigée. Pull request #18175
  • La position du champ Multiplicateur d'échelle dans l'interface utilisateur a été améliorée (Draft SetStyle, Draft Éditer le style des annotations et Draft Préférences). Pull request #18299

Atelier FEM

Autres améliorations de FEM

  • La verbosité des logs peut maintenant être définie pour Gmsh dans les préférences. Pull request #17699
  • La propriété DonnéesSecond Order Linear et la prise en charge du maillage plus fin localement, jusque là disponibles uniquement pour Gmsh, sont maintenant également disponibles pour la nouvelle implémentation de Netgen. Pull request #17170
  • Les types de sections de poutres-caissons et elliptiques ont été ajoutés à FEM Coupe transversale d'un élément 1D. Pull request #15843
  • L'outil Purger les résultats supprime désormais tous les objets résultat et pas seulement ceux natifs de CalculiX. Pull request #18328
  • La contrainte de liaison peut maintenant être appliquée également aux faces de coques. Pull request #18325
  • Le format de sortie (binaire ou ASCII) et la sauvegarde des ID de géométrie peuvent maintenant être définis pour Elmer, de même dans les préférences. Pull request #17972

Atelier Material

Autres améliorations de Material

  • Les propriétés physiques basées sur le matériau sont désormais affichées dans la vue des propriétés. Il s'agit du nom du matériau, de sa densité, de sa masse et de son volume. La densité par défaut est de 1 kg/m3. Pull request #17438

Atelier Sketcher


Cliquez sur l'image si l'animation ne démarre pas. 		L'outil Projection d'arêtes a été ajouté, ce qui permet de créer une géométrie géométrie externe de définition et de basculer entre les modes de définition et de construction pour la géométrie externe.

Pull request #17736

L'outil Intersection d'arêtes a été ajouté, ce qui permet de créer une géométrie externe basée sur la géométrie de référence sélectionnée et l'intersection du plan de l'esquisse.

Pull request #17736


Cliquez sur l'image si l'animation ne démarre pas. 		Les géométries externes (projection et intersection) peuvent désormais être créées en sélectionnant une face.

Pull request #17736

Autres améliorations de Sketcher

  • Il est maintenant possible d'utiliser directement une géométrie externe comme entrée pour des outils comme le décalage ou la transformation (réseau), pour les deux géométries externes construction et définition Pull request #17615.
  • Les axes de Sketcher sont désormais affichés avec une longueur infinie. Pull request #17312
  • Les esquisses sont désormais classées par ordre alphabétique dans la boîte de dialogue de Ancrer une esquisse. Pull request #16518
  • Le déplacement de groupe a été ajouté, ce qui permet de déplacer toutes les entités géométriques sélectionnées. Pull request #18273

Atelier Spreadsheet

Autres améliorations de Spreadsheet

  • Des raccourcis par défaut pour Texte en gras, Texte en italique et Texte en souligné ont été ajoutés. Pull request #15556
  • Double-cliquer sur le séparateur dans l'en-tête redimensionne maintenant la colonne en fonction du contenu. Pull request #16296
