Description

Exporte un fichier de programme de robot. L'exportation peut être soit un sous-programme compressé KUKA, soit un sous-programme complet KUKA.

Utilisation

  1. Sélectionnez le robot et la trajectoire dans la Vue en arborescence.
  2. Il existe plusieurs façons d'exporter la trajectoire :
    • Choisissez Robot → Exporter la trajectoire → Sous-programme compressé Kuka... dans le menu supérieur.
    • Choisissez Robot → Exporter la trajectoire → Sous-programme complet Kuka... dans le menu supérieur.

Résultat : un fichier KRL (*.src) sera créé.
