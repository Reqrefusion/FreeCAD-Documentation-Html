Introduction

L'atelier Web n'est pas vraiment un atelier de modélisation mais fournit une fenêtre de navigateur Web dans FreeCAD.

À partir de ce navigateur, vous pouvez naviguer sur le Web, cliquer sur des liens, lire la documentation en ligne et vous connecter à des sites Web (les cookies sont pris en charge). L'atelier Web constitue également une base pour le développement d’outils Web dans FreeCAD. La fenêtre du navigateur est basée sur QtWebEngine et fournit des boutons de navigation standard tels que arrêter, recharger, pages précédente et suivante, ainsi que les zooms avant et arrière.

Outils

L'atelier Web offre les outils suivants :

  • Définir l'URL : modifie l'URL de l'onglet web actif.
  • Ouvrir le site web... : ouvre le site web de FreeCAD.
    • Page précédente : permet d'accéder à la page web précédente.
  • Page suivante : permet d'accéder à la page web suivante.
  • Actualiser la page web : met à jour la page web courante.
  • Arrêter le chargement : arrête le chargement de la page web en cours.
  • Zoom avant : effectue un zoom avant sur la page web en cours.
  • Zoom arrière : effectue un zoom arrière sur la page web en cours.
