Description

La commande SheetMetal Forme de base crée un objet SheetMetal BaseShape à partir de paramètres.

Les cinq formes de base disponibles : forme en L, forme en U, forme en baignoire, forme en chapeau et boîte

Une sixième forme rectangulaire, appelée Plat, a été introduite dans la version 0.4.10.

Utilisation

  1. Activez la commande SheetMetal Forme de base en utilisant l'une des méthodes suivantes :
    • Le bouton Créer une forme de base.
    • L'option de menu SheetMetal → Créer une forme de base.
    • Le raccourci clavier : H.
  2. Le panneau des tâches Générer une forme de base de la tôle s'ouvre.
  3. Sélectionnez la forme désirée dans les options Type de forme de base.
  4. Ajustez les paramètres.
  5. Appuyez sur OK pour terminer la commande.

Propriétés

Un objet SheetMetal BaseShape est dérivé d'un objet Part Feature ou, s'il se trouve à l'intérieur d'un PartDesign Corps, à partir d'un objet PartDesign Feature, dont il hérite de toutes les propriétés. Il possède également les propriétés supplémentaires suivantes :

Données

Parameters

  • Donnéesfill Gaps (Bool) : prolonge les côtés et les rebords pour combler tous les espaces. Valeur par défaut : true.
  • DonnéesflangeWidth (Length) : largeur du rebord supérieure. Valeur par défaut : 5,00 mm.
  • Donnéesheight (Length) : hauteur de la forme. Valeur par défaut : 10,00 mm.
  • Donnéeslength (Length) : longueur de la forme. Valeur par défaut : 30,00 mm.
  • Donnéesorigin Loc (Enumeration) : position de l'origine.
    -X,-Y, -X,0, -X,+Y, 0,-Y, 0,0 (valeur par défaut), 0,+Y, +X,-Y, +X,0 et +X,+Y.
  • Donnéesradius (Length) : rayon de courbure. Valeur par défaut : 1,00 mm.
  • Donnéesshape Type (Enumeration) : type de forme de base.
    Flat (introduite dans la V0.4.10), L-Shape (valeur par défaut), U-Shape, Tub, Hat et Box.
  • Donnéesthickness (Length) : épaisseur de la tôle. Valeur par défaut : 1,00 mm.
  • Donnéeswidth (Length) : largeur de la forme. Valeur par défaut : 20,00 mm.
