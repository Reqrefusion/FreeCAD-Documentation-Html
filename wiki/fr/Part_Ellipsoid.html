Description

Un Part Ellipsoïde est un solide paramétrique qui peut être créé avec la commande Part Primitives. Dans le système de coordonnées défini par sa propriété DonnéesPlacement, les axes de l'ellipsoïde sont alignés sur les axes X, Y et Z, et son centre est donc positionné à l'origine.

Un Part Ellipsoïde peut être tronqué en haut et/ou en bas en modifiant ses propriétés DonnéesAngle1 et/ou DonnéesAngle2. Il peut être transformé en un segment d'ellipsoïde en modifiant sa propriété DonnéesAngle3.

Propriétés

Voir aussi : Éditeur de propriétés

Un objet Part Ellipsoïde est dérivé d'un Part Feature et hérite de toutes ses propriétés. Il possède également les propriétés supplémentaires suivantes :

Données

Attachment

L'objet a les mêmes propriétés d'attachement qu'un Part Part2DObject.

Ellipsoid

  • DonnéesRadius1 (Length) : rayon de l'ellipsoïde dans sa direction Z. La valeur par défaut est 2mm.
  • DonnéesRadius2 (Length) : rayon de l'ellipsoïde dans sa direction X. La valeur par défaut est 4mm.
  • DonnéesRadius3 (Length) : rayon de l'ellipsoïde dans sa direction Y. La valeur par défaut est 4mm. La valeur par défaut est 4mm.
  • DonnéesAngle1 (Angle) : angle de départ des côtés elliptiques de l'ellipsoïde. Plage valide : -90° <= value < 90°. Doit être inférieur à DonnéesAngle2. La valeur par défaut est -90°.
  • DonnéesAngle2 (Angle) : angle de fin des côtés elliptiques de l'ellipsoïde. Plage valide : -90° < value <= 90°. Doit être supérieur à DonnéesAngle1. La valeur par défaut est 90°. Si l'angle total des côtés de l'ellipse est inférieur à 180°, l'ellipsoïde sera tronqué et aura une face plate en haut et/ou en bas.
  • DonnéesAngle3 (Angle) : angle total de l'ellipsoïde dans son plan XY. Plage valide : 0° < value <= 360°. La valeur par défaut est 360°. Si elle est inférieure à 360°, le solide résultant sera un segment d'ellipsoïde.

Script

Voir aussi : Autogenerated API documentation, Part Ecrire un script et Débuter avec les scripts FreeCAD.

Un Part Ellipsoïde est créé avec la méthode addObject() du document : 

ellipsoid = FreeCAD.ActiveDocument.addObject("Part::Ellipsoid", "myEllipsoid")
  • Où "myEllipsoid" est le nom de l'objet.
  • La fonction restitue l'objet nouvellement créé.

Exemple : 

import FreeCAD as App

doc = App.activeDocument()

ellipsoid = doc.addObject("Part::Ellipsoid", "myEllipsoid")
ellipsoid.Radius1 = 2
ellipsoid.Radius2 = 4
ellipsoid.Radius3 = 6
ellipsoid.Angle1 = -90
ellipsoid.Angle2 = 50
ellipsoid.Angle3 = 300
ellipsoid.Placement = App.Placement(App.Vector(1, 2, 3), App.Rotation(15, 0, 20))

doc.recompute()
