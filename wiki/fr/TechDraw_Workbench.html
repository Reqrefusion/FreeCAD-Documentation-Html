Introduction

L' atelier TechDraw est utilisé pour produire des dessins techniques de base à partir de modèles 3D créés avec un autre atelier, tels que Part, PartDesign ou BIM, ou importés à partir d'autres applications. Chaque dessin est une page pouvant contenir diverses vues d'objets pouvant être dessinés, telles que Part::Features, PartDesign::Bodies, App::Part groups et des groupes Object Document. Les dessins résultants peuvent être utilisés pour des éléments tels que la documentation, les instructions de fabrication, les contrats, les permis, etc.

Des dimensions, des sections, des zones hachurées, des annotations et des symboles SVG peuvent être ajoutés à la page, qui peuvent ensuite être exportés vers différents formats tels que DXF, SVG et PDF.

Si votre objectif principal est la production de dessins 2D complexes et de fichiers DXF et que vous n'avez pas besoin de modélisation 3D, FreeCAD n'est peut-être pas le bon choix pour vous. Vous pouvez envisager d'utiliser un logiciel dédié au dessin technique, tel que LibreCAD ou QCad.

Aimantation

introduit dans la version 1.0 : l'atelier TechDraw dispose d'une fonction d'aimantation. Elle peut être utilisée pour aligner automatiquement les vues, les vues de section et les cotes lorsqu'on les positionne en les faisant glisser avec la souris. Avec l'option Aimanter les vues à l'alignement activée (par défaut) dans les préférences, les vues s'aligneront avec d'autres vues lorsqu'elles sont suffisamment proches (paramètre Coefficient d'aimantation des vues). Les cotes sont également alignées sur d'autres cotes parallèles et le texte de la cote peut être aligné sur le milieu de la ligne de cote. L'aimantation peut être temporairement désactivé en maintenant la touche Alt enfoncée.

Outils

Pages

  • Page par défaut : ajoute une nouvelle page en utilisant le modèle par défaut.
  • Page selon un modèle : ajoute une nouvelle page en utilisant un modèle sélectionné.
  • Mettre à jour les champs du modèle : introduit dans la version 1.0
  • Rafraîchir : force la mise à jour de la page.
  • Tout imprimer : imprime toutes les pages d'un document. introduit dans la version 0.21
  • Sauvegarder au format SVG : enregistre la page en cours sous forme de fichier SVG.
  • Sauvegarder au format DXF : enregistre la page en cours sous forme de fichier DXF.

Vues

Vues de TechDraw

  • Vue : ajoute une représentation d'un ou plusieurs objets. introduit dans la version 1.0 : permet de créer une vue unique, un groupe de projection, une vue de l'atelier Spreadsheet, une vue de l'atelier Arch, un symbole ou une vue d'une image.
  • Vue interrompue : ajoute une vue interrompue d'un ou plusieurs objets. introduit dans la version 1.0
  • Vue en coupe : insère une vue en coupe à partir d'une vue existante.
  • Vue en coupe complexe : insère une vue en coupe d'une vue existante en fonction d'un profil. introduit dans la version 0.21
  • Vue détaillée : ajoute une vue détaillée d'une partie d'une vue existante.
  • Groupe de projections : ouvre une boîte de dialogue pour créer plusieurs vues d'un objet depuis différentes directions. introduit dans la version 1.0 : l'outil Vue peut être utilisé à la place.
  • Fenêtre de rognage : insère une fenêtre de rognage.
  • Symbole : insère un symbole SVG dans une page. introduit dans la version 1.0 : l'outil Vue peut être utilisé à la place.
  • Image : insère une image bitmap PNG ou JPG dans une page. introduit dans la version 1.0 : l'outil Vue peut être utilisé à la place.
  • Copier une vue : partage une vue entre plusieurs pages.
  • Bascule des cadres : bascule l'affichage des cadres et des étiquettes entourant une vue.
  • Projection de forme : crée des projections de formes dans la vue 3D.

Vues d'autres ateliers

  • Vue active : insère une vue de la vue 3D active.
  • Vue d'un objet Draft : ajoute une vue d'un objet Draft.
  • Vue d'un objet BIM : ajoute une vue d'un objet Arch Plan de coupe de l'atelier BIM. introduit dans la version 1.0 : l'outil Vue peut être utilisé à la place.
  • Vue d'un objet Spreadsheet : insère une vue d'une feuille de calcul de l'atelier Spreadsheet. introduit dans la version 1.0 : l'outil Vue peut être utilisé à la place.

Empilement

Il s'agit d'outils permettant de modifier l'ordre d'empilement qui contrôle la profondeur apparente des vues sur une page.

  • Ajuster l’ordre d’empilage des vues :
  • Empiler en haut : déplace les vues au sommet de l'ordre d'empilement. introduit dans la version 0.21
  • Empiler en bas : déplace les vues au bas de l'ordre d'empilement. introduit dans la version 0.21
  • Empiler vers le haut : déplace les vues d'un niveau supérieur dans l'ordre d'empilement. introduit dans la version 0.21
  • Empiler vers le bas : déplace les vues vers le bas d'un niveau dans l'ordre d'empilement. introduit dans la version 0.21

Cotes

  • Insérer une cote :
  • Cote : ajoute une cote contextuelle. introduit dans la version 1.0
  • Cote de longueur : ajoute une cote de longueur.
  • Cote horizontale : ajoute une cote de longueur horizontale.
  • Cote verticale : ajoute une cote de longueur verticale.
  • Cote de rayon : ajoute une cote de rayon à un cercle ou un arc de cercle.
  • Cote de diamètre : ajoute une cote de diamètre à un cercle ou à un arc de cercle.
  • Cote angulaire : ajoute une cote angulaire entre deux arêtes droites.
  • Cote angulaire par 3 points : ajoute une cote d'angle à partir de trois sommets.
  • Cote de surface : ajoute une cote de surface à une face. introduit dans la version 1.0
  • Longueur d'arc : crée une cote de longueur d'arc.
  • Cote étendue horizontale : ajoute une cote étendue horizontale.
  • Cote étendue verticale : ajoute une cote étendue verticale.
  • Cotes horizontales : crée une séquence horizontale de cotes.
  • Cotes verticales : crée une séquence verticale de cotes.
  • Cotes inclinées : crée une séquence de cotes inclinées alignées.
  • Cotes parallèles horizontales : crée des cotes horizontales uniformément espacées à partir de la même ligne de base.
  • Cotes parallèles verticales : crée des cotes verticales uniformément espacées à partir de la même ligne de base.
  • Cotes parallèles inclinées : crée des cotes inclinées uniformément espacées à partir de la même ligne de base.
  • Cote horizontale chanfrein : crée une cote horizontale et une cote d'angle pour un chanfrein.
  • Cote verticale chanfrein : crée une cote verticale et une cote d'angle pour un chanfrein.
  • Infobulle : ajoute une "infobulle" dans la page.
  • Cote axonométrique : ajoute une cote axonométrique. introduit dans la version 0.21
  • Cote à partir des points du repère - EXPÉRIMENTAL : ajoute une cote linéaire à partir des points du repère.
  • Réparation des cotes : permet d'ajuster les références géométriques 2D ou 3D d'une cote. introduit dans la version 0.21
  • Lier une cote : lie une cote existante à la géométrie 3D (obsolète).

Hachures

  • Hachures par motif : hachure une face en utilisant un fichier image.
  • Applique un motif de hachure : applique un motif de hachure à une face en utilisant une spécification Autodesk PAT.

Annotations

  • Annotation : ajoute un bloc de texte pour servir d'annotation.
  • Ligne de repère : ajoute une ligne de repère à une vue.
  • Annotation texte enrichi : ajoute un bloc d’annotation en texte enrichi à une ligne de repère ou à une vue.
  • Ajouter des sommets cosmétiques :
  • Point cosmétique : ajoute un sommet qui ne fait pas partie de la géométrie principale.
  • Points médians : ajoute des points cosmétiques aux points médians d'une ou de plusieurs arêtes sélectionnées.
  • Quadrant : ajoute des points cosmétiques tous les quarts de points d'un ou de plusieurs bords sélectionnés (circulaires).
  • Ajouter des lignes centrales :
  • Ligne centrale à une face : ajoute une ligne centrale aux face(s) sélectionnée(s).
  • Ligne centrale entre 2 lignes : ajoute une ligne centrale entre 2 arêtes.
  • Ligne centrale entre 2 points : ajoute une ligne centrale entre 2 points.
  • Ligne cosmétique par 2 points : ajoute une ligne cosmétique reliant 2 points.
  • Cercle cosmétique : ajoute un cercle cosmétique. introduit dans la version 1.0
  • Apparence des lignes : modifie l'apparence des lignes.
  • Montrer tout : affiche/masque les lignes/arêtes invisibles dans une vue.
  • Soudure : ajoute des spécifications de soudage à une ligne de repère existante.
  • Symbole d'état de surface : ajoute un symbole de finition de surface à une page. introduit dans la version 0.21
  • Tolérance de trou/d'arbre : ajoute les tolérances de trous ou d'arbres selon la norme ISO 286 à une cote. introduit dans la version 0.21

Extensions

Attributs et modifications

  • Choix des attributs : sélectionne les attributs (style, largeur et couleur) des nouvelles lignes cosmétiques et des lignes centrales, et spécifie l'espacement en cascade et la distance delta.
  • Modification des attributs : modifie les attributs (style, largeur et couleur) des lignes cosmétiques et des lignes centrales.
  • Modifier la longueur des lignes cosmétiques ou des lignes centrales :
  • Prolonger : prolonge une ligne cosmétique ou une ligne centrale aux deux extrémités.
  • Raccourcir : raccourcit une ligne cosmétique ou une ligne centrale aux deux extrémités.
  • Verrouiller/déverrouiller : verrouille/déverrouille la position d'une vue.
  • Positionner une vue en coupe : aligne orthogonalement une vue en coupe avec sa vue source.
  • Aligner des cotes :
  • Aligner horizontalement : aligne des cotes horizontales pour créer une chaîne de cotes.
  • Aligner verticalement : aligne des cotes verticales pour créer une chaîne de cotes.
  • Aligner obliquement : aligne des cotes inclinées pour créer une chaîne de cotes.
  • Espacements réguliers :
  • Cascade horizontale : espace régulièrement des cotes horizontales.
  • Cascade verticale : espace régulièrement des cotes verticales.
  • Cascade inclinée : espace régulièrement des cotes inclinées.
  • Surface : calcule la surface des faces sélectionnées et insère une annotation de surface.
  • Longueur d'arc d'arêtes : calcule la longueur d'arc d'arêtes sélectionnées et insère une annotation de longueur d'arc. introduit dans la version 1.0
  • Personnaliser l'infobulle : personnalise le formatage du texte d'une infobulle ou du texte d'une cote. Des symboles GD&T et d'autres caractères spéciaux peuvent être ajoutés.

Lignes centrales, filetage et taraudage

  • Ajouter des axes de centrage :
  • Axes de centrage : ajoute des lignes de centre aux cercles et aux arcs.
  • Axes de centrage de trous/vis : dessine les lignes centrales des cercles de trous/vis.
  • Ajouter des filetages/taraudages cosmétiques :
  • Corps de taraudage : ajoute un symbole pour le taraudage à la vue latérale d'un trou.
  • Taraudage : ajoute des taraudages symboliques à la vue supérieure ou inférieure des trous.
  • Corps de filetage : ajoute un symbole pour le filetage à la vue latérale des boulons/vis/tiges.
  • Filetage : ajoute des filetages symboliques à la vue supérieure ou inférieure des boulons/vis/tiges.
  • Ajoute des sommets :
  • Intersection de lignes : crée des sommets cosmétiques à l'intersection des lignes sélectionnées.
  • Ajoute un sommet décalé : ajoute un sommet cosmétique à un décalage spécifié par rapport à un sommet sélectionné. introduit dans la version 1.0
  • Ajouter des cercles ou des arcs cosmétiques :
  • Cercle : dessine un cercle cosmétique en utilisant deux points.
  • Arc : ajoute un arc cosmétique dans le sens inverse des aiguilles d'une montre à partir de trois sommets.
  • Cercle par 3 points : ajoute un cercle cosmétique basé sur trois sommets.
  • Ajouter des lignes cosmétiques parallèles ou perpendiculaires :
  • Ligne parallèle : dessine une ligne cosmétique parallèle à une autre ligne passant par un sommet.
  • Ligne perpendiculaire : dessine une ligne cosmétique perpendiculaire à une autre ligne passant par un sommet.

Cotes

Plusieurs des outils de cotation des extensions sont répertoriés dans la section Cotes ci-dessus.

  • Insérer un préfixe :
  • Diamètre : insère un symbole "⌀" pour un diamètre au début du texte de la cote.
  • Section tube : insère un symbole "□" pour une section de tube carré au début du texte de la cote.
  • Symbole de répétition : insère un symbole de répétition au début du texte de la cote. introduit dans la version 1.0
  • Supprimer les symboles : supprime tous les symboles devant le texte de la cote.
  • Modifier les décimales :
  • Plus de décimales : augmente le nombre de décimales du texte de la cote.
  • Moins de décimales : diminue le nombre de décimales du texte de la cote.

Divers

  • Gomme : supprime les objets cosmétiques d'une page.

Outils obsolètes

  • Ajout vue à une fenêtre de rognage : ajoute une vue existante à une fenêtre de rognage. Non disponible dans version 1.0 et suivantes.
  • Suppression vue d'une fenêtre de rognage : supprime une vue d'une fenêtre de rognage. Non disponible dans version 1.0 et suivantes.
  • Déplacer une vue : déplace une vue et ses dépendants vers une page différente. Non disponible dans version 1.0 et suivantes.

Fonctions supplémentaires

  • Groupes de lignes : pour contrôler l'apparence de divers types de lignes.
  • Modèles : modèles par défaut définis pour les pages de dessin.
  • Hachures : explication des différentes techniques de hachurage.
  • Dimensionnement géométrique et tolérancement : explication sur la façon de réaliser le dimensionnement géométrique et le tolérancement.

Préférences

  • Préférences : préférences pour les valeurs par défaut de la page de dessin, telles que l'angle de projection, les couleurs, la taille du texte et les styles de trait.

Limitations

  • Ne pas couper, copier et coller des objets TechDraw dans la vue en arborescence car cela ne fonctionne généralement pas bien.
  • Ne pas faire glisser les objets TechDraw dans la vue en arborescence avec la souris.
