Description

Définit les valeurs par défaut pour la création de points de passage.

Utilisation

  1. Choisissez Robot → Définir les valeurs par défaut dans le menu supérieur, pour définir :
  • vitesse
  • continu (mode)
  • accélération

Remarques

Les valeurs par défaut du programme sont les suivantes :

  • vitesse = 1 m/s
  • continu = false
  • accélération = 1 m/s^2
