Description

La commande Draft Appliquer le style applique les paramètres de style en cours aux objets sélectionnés.

version 0.21 et précédentes : cette commande ne gère que cinq des paramètres offerts par la commande Draft Définir le style.

introduit dans la version 0.21 : cette commande modifie les propriétés d'affichage des objets. Elle applique tous les paramètres offerts par la commande Draft Définir le style. Elle modifie également ces autres propriétés :

  • VueDecimals (pour les dimensions) : Voir Draft Préférences.
  • VueShowLine (pour les dimensions) : Idem.

Utilisation

  1. Vous pouvez modifier les paramètres du style avec la commande Draft Définir le style.
  2. Sélectionnez un ou plusieurs objets.
  3. Sélectionnez l'option Utilitaires → Appliquer le style en cours du menu, ou dans le menu contextuel de la vue en arborescence ou de la vue 3D.

Remarques

  • La commande Draft Définir le style peut également appliquer les paramètres.
