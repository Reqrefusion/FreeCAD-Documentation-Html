Description

La commande Std Mesurer donne accès à l'outil de mesure unifié. Elle remplace plusieurs commandes de mesure antérieures, offrant ainsi une fonctionnalité de mesure polyvalente.

Utilisation

  1. introduit dans la version 1.1 : vous pouvez appuyez sur le bouton pour changer les paramètres :
    • Sauvegarde automatique : si la case est cochée, la dernière mesure est sauvegardée lors du démarrage d'une nouvelle mesure (maintenir Maj peut temporairement désactiver ce comportement),
    • Sélection additive : si la case est cochée, cliquez sur des entités géométriques successives les ajoute en tant que sélections pour la même mesure, sinon Ctrl doit être pressé pour le faire.
  2. Vous pouvez présélectionner les entités à mesurer.
  3. Appuyez sur le bouton Mesurer ou sélectionnez l'option Outils → Mesurer du menu.
  4. Si aucune entité géométrique n'a été présélectionnée, effectuez l'une des opérations suivantes :
    • Sélectionnez les entités géométriques tout en laissant le Mode à Automatique pour que le mode soit automatiquement déterminé en fonction de la sélection.
    • Choisissez le Mode autre que Automatique et sélectionnez ensuite les entités géométriques (un nouveau clic sur celles-ci les désélectionnera) :
      • Distance : distance la plus courte entre les entités sélectionnées,
      • Distance libre : distance entre deux points librement sélectionnés (appartenant à des entités différentes telles que des arêtes, des faces). Si des arêtes circulaires sont sélectionnées, la distance entre les centres des cercles est indiquée.
      • Angle : angle entre les entités sélectionnées,
      • Longueur : longueur de l'arête sélectionnée,
      • Position : coordonnées du sommet sélectionné,
      • Surface : surface de la face sélectionnée,
      • Rayon : rayon de l'arête circulaire sélectionnée,
      • Centre de masse : centre de masse de l'arête, de la face ou du solide sélectionné (seulement si présélectionné dans l'arborescence).
  5. Le résultat de la mesure sera affiché dans le champ Résultat et sur une étiquette affichée dans la vue_3D. Cette étiquette comprendra également une icône indiquant le type de mesure. Les étiquettes peuvent être personnalisées dans le réglage des préférences. Elles peuvent être déplacées dans la vue 3D en les faisant glisser à l'aide d'un curseur.
  6. Cliquez sur Réinitialiser pour supprimer la mesure ou sur Enregistrer pour la conserver (les mesures sauvegardées seront conservées dans le conteneur Measurements de la vue en arborescence).
  7. Cliquez sur Fermer pour quitter la commande.

Options

  • Show Delta : active ou désactive la propriété DonnéesShow Delta (affichage des mesures projetées) utilisée pour les modes Distance et Distance Free. Activée par défaut.

Propriétés

Vue

Appearance

  • DonnéesFont Size (Integer) : taille de la police pour l'étiquette de la cote sauvegardée.
  • DonnéesLine Color (Color) : couleur de la ligne de la cote affichée dans la vue 3D.
  • DonnéesShow Delta (Bool) : si true, les mesures de distance projetées sont affichées dans la vue 3D. Disponible uniquement pour les mesures Distance et Distance libre. Les valeurs de mesure projetées sont également toujours accessibles dans le groupe Measurement de l'onglet Données de la propriété.
  • DonnéesText Background Color (Color) : couleur de l'arrière-plan de l'étiquette de la cote.
  • DonnéesText Color (Color) : couleur du texte et du symbole de l'étiquette de la cote.

Remarques

  • Cette commande est le résultat d'un projet GSoC de 2023 (En).
