Description

L'outil TechDraw Vue en coupe complexe insère une vue en coupe transversale basée sur une vue de la pièce existante et un profil.

Une vue en quart de coupe créée avec l'outil Vue en coupe complexe

Une vue en coupe alignée créée avec l'outil Vue en coupe complexe

Une vue en coupe décalée créée avec l'outil Vue en coupe complexe

Utilisation

  1. Sélectionnez une vue de la pièce et un objet de profil dans la vue 3D ou la vue en arborescence. Les profils sont généralement des esquisses, mais tout objet dont la forme peut être transformée en polyligne fonctionnera.
  2. Il existe plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Insérer une vue en coupe complexe.
    • Sélectionnez l'option TechDraw → Vues de Techdraw → Insérer une vue en coupe complexe dans le menu.
  3. Un panneau de tâches s'ouvre et vous aide à calculer les différentes propriétés. Des valeurs raisonnables pour la direction de la vue sont calculées, mais elles peuvent être modifiées.

Propriétés

Une vue en coupe complexe, en fait un objet TechDraw::DrawComplexSection, est dérivée d'une vue en coupe, objet TechDraw::DrawViewSection, et hérite de toutes ses propriétés. Elle possède également les propriétés supplémentaires suivantes :

Données

Cutting Tool

  • DonnéesCutting Tool Wire Object : l'objet document dont la forme sera utilisée pour générer le profil de coupe.
  • DonnéesProjection Strategy : contrôle la façon dont la découpe est effectuée et comment le résultat est projeté sur la page :
    • Offset : effectue une découpe simple de la forme source et projette le résultat.
    • Aligned : découpe la forme source à l'aide d'un outil créé à partir de chaque segment (bord) du profil de découpe. Les résultats de chaque coupe sont projetés dans un tableau vertical ou horizontal, selon l'orientation du profil de coupe.
    • NoParallel : comme Aligned, mais les segments de profil qui sont parallèles à la direction de la vue sont ignorés.

Script

Voir aussi : Autogenerated API documentation et FreeCAD Débuter avec les scripts.

Une Vue en coupe complexe peut être créée à partir de macros et à partir de la console Python à l'aide des fonctions suivantes : 

doc = FreeCAD.ActiveDocument
box = doc.Box
profile = doc.Sketch
page = doc.Page

view = doc.addObject("TechDraw::DrawViewPart", "View")
page.addView(view)
view.Source = box
view.Direction = (0, 0, 1)

section = doc.addObject("TechDraw::DrawComplexSection", "ComplexSection")
page.addView(section)
section.BaseView = view
section.CuttingToolWireObject = profile
section.Direction = (0, 1, 0)
section.SectionNormal = (-1, 0, 0)
