Description

L'outil Copier crée une copie du parcours sélectionné.

Utilisation

  1. Sélectionnez un objet de l' atelier CAM
  2. Il y a plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Copier l'opération dans la tâche dans la barre d'outils CAM
    • Utilisez le raccourci clavier P puis Y
    • Utilisez l'entrée CAM → Modification du parcours → Copier l'opération dans la tâche depuis le menu CAM

Propriétés

  • DonnéesBase : parcours de base devant être copié

Options

La copie reste liée au parcours original. Si l'original change, la copie aussi.
