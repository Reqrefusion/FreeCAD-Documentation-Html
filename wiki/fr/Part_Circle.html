Description

Un Part Cercle est une forme paramétrique qui peut être créée avec la commande Part Primitives. Dans le système de coordonnées défini par sa propriété DonnéesPlacement, le cercle se trouve sur le plan XY avec son centre à l'origine.

Un Part Cercle est en fait un arc de cercle fermé dans le sens inverse des aiguilles d'une montre. Il peut être transformé en arc de cercle en modifiant ses propriétés DonnéesAngle1 et/ou DonnéesAngle2.

Utilisation

Voir Part Primitives.

Un Part Cercle peut également être créé en sélectionnant trois points :

  1. Dans le panneau des tâches de la commande Créer des primitives, sélectionnez l'option Cercle dans la liste déroulante.
  2. Appuyez sur le bouton À partir de trois points.
  3. Sélectionnez trois sommets dans la vue 3D. Il n'est pas nécessaire de maintenir la touche Ctrl enfoncée.
  4. Un cercle est créé.
  5. Les sommets sélectionnés ne sont utilisés qu'au moment de la création pour calculer les DonnéesRadius et DonnéesPlacement du cercle.

Propriétés

Voir aussi: Éditeur de propriétés

Un objet Part Cercle est dérivé d'un Part Feature et hérite de toutes ses propriétés. Il possède également les propriétés supplémentaires suivantes :

Données

Attachment

L'objet a les mêmes propriétés d'attachement qu'un Part Part2DObject.

Base

  • DonnéesRadius (Length) : rayon du cercle ou de l'arc de cercle. La valeur par défaut est 2mm.
  • DonnéesAngle1 (Angle) : angle de départ de l'arc de cercle. Plage valide : 0° < value <= 360°. La valeur par défaut est 0°.
  • DonnéesAngle2 (Angle) : angle final de l'arc de cercle. Plage valide : 0° < value <= 360°. La valeur par défaut est 360°. Si DonnéesAngle1 et DonnéesAngle2 sont égaux, ou si un angle est 0° et l'autre 360°, un cercle complet est créé.

Script

Voir aussi : Autogenerated API documentation, Part Ecrire un script et Débuter avec les scripts FreeCAD.

Un Part Cercle est créé avec la méthode addObject() du document : 

circle = FreeCAD.ActiveDocument.addObject("Part::Circle", "myCircle")
  • Où "myCircle" est le nom de l'objet.
  • La fonction restitue l'objet nouvellement créé.

Exemple : 

import FreeCAD as App

doc = App.activeDocument()

circle = doc.addObject("Part::Circle", "myCircle")
circle.Radius = 10
circle.Angle1 = 45
circle.Angle2 = 225
circle.Placement = App.Placement(App.Vector(1, 2, 3), App.Rotation(30, 45, 10))

doc.recompute()
