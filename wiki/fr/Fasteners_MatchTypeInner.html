Description

La commande Fasteners Adapter pour le trou taraudé se comporte comme une bascule avec la commande Fasteners Adapter pour le trou de passage. Lorsque la commande Fasteners Adapter pour le trou taraudé est sélectionnée, les bords circulaires sont considérés comme des trous taraudés lorsque de nouvelles fixations y sont fixées. Cela détermine les dimensions par défaut de ces fixations. Ce paramètre n'est pas utilisé pour les vis à tête fraisée.

Utilisation

  1. Il existe plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Adapter pour le trou taraudé.
    • Sélectionnez l'option Fasteners → Adapter pour le trou taraudé du menu.
  2. Le bouton reste enfoncé pour indiquer l'option sélectionnée.
