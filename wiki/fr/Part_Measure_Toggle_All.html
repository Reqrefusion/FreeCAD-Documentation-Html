Description

Cette commande permet de basculer la visibilité de toutes les mesures dans la vue 3D.

Utilisation

  1. Il y a plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Tout basculer.
    • Sélectionnez l'option Mesure → Tout basculer dans le menu.
