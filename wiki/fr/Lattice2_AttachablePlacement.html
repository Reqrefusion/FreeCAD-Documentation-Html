Description

La commande Lattice2 AttachablePlacement attribue un placement ou une poignée à un objet afin qu'il puisse plus tard être référencé par un emplacement.

Utilisation

  1. Basculez vers l'atelier externe Lattice2
  2. Appelez la commande Placement attaché de plusieurs manières:
    • Appuyez sur l'icône dans la barre d'outils Lattice2
    • Utilisez la Lattice2 → Attached Placement → Attached Placement dans la liste déroulante Lattice2
    • Utilisez le raccourci clavier {{{1}}} {{{1}}}
  3. Choisissez les options du menu de dialogue

Propriétés

Remarques

  • Le placement doit être fait à l'intérieur du corps.
