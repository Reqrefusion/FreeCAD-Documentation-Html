Description

La Draft Barre permet de sélectionner le plan de travail, de définir les paramètres de style, de basculer en mode construction et de spécifier le calque ou le groupe actif. Les quatre boutons affichent les paramètres en cours. La Draft Barre est disponible dans l' atelier Draft et l' atelier BIM et ne peuvent pas être désactivés.

La Draft Barre

Outils disponibles

  • Plan de travail : sélectionne le plan de travail en cours de Draft.
  • Définir le style : définit le style par défaut pour les nouveaux objets.
  • Basculer en mode construction : active ou désactive le mode de construction de Draft.
  • Groupement automatique : change le Draft Calque actif ou, accessoirement, le Std Groupe actif ou un objet de type groupe de BIM.
