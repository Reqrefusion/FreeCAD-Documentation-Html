Description

La commande Draft Basculer en mode construction active ou désactive le mode de construction de Draft. Si le mode construction est activé, les nouveaux objets Draft sont placés dans un groupe dédié et reçoivent une couleur prédéfinie. Cette fonction est destinée à la géométrie de construction, souvent temporaire, utilisée pour fournir de nouveaux points d'aimantation pour la création d'autres objets. Lorsque la géométrie de construction n'est plus nécessaire, le groupe de construction peut facilement être caché ou effacé.

Géométrie de construction, en bleu, utilisée pour déterminer le centre et le rayon d'un cercle

Utilisation

  1. Il y a plusieurs façons de lancer la commande :
    • Appuyez sur le bouton de la Draft Barre. Ce bouton est activé si le mode de construction Draft est en cours.
    • Draft : sélectionnez l'option Utilitaires → Basculer en mode construction du menu ou dans le menu contextuel de la vue en arborescence ou de la vue 3D.
    • Draft: utilisez le raccourci clavier : C puis M.
  2. Le bouton de la Draft Barre est mis à jour.

Remarques

  • Si le mode de construction Draft est activé, le calque actif est ignoré.

Préférences

  • Pour modifier l'étiquette du groupe de construction : Édition → Préférences... → Draft → Général → Étiquette du groupe de construction.
  • Pour modifier la couleur utilisée : Édition → Préférences... → Draft → Général → Couleur de la géométrie de construction.
