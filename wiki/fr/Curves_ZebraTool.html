Description

L' outil Zebra crée une texture Zebra pour l'inspection de surface. Cet outil fait partie des ateliers externes appelé Curves.

Utilisation

  1. Passer à l'atelier Curves (à installer à partir du Gestionnaire des extensions si ce n'est pas déjà fait)
  2. Sélectionnez la surface à tester
  3. Pour appeler la commande, effectuez l'une des opérations suivantes:
    • Appuyez sur le bouton Zebra texture for surface inspection dans la barre d'outils.
    • Utilisez le Surfaces → Zebra Tool
  4. Modifiez le paramètre d'Outil Zebra aux conditions requises

Propriétés

  • DonnéesBlack Stripes Width : modifie la largeur des bandes noires.
  • DonnéesScale : redimensionne les rayures zébrées.
  • DonnéesRotation : fait pivoter les rayures zébrées.

Remarques

  • Cliquez sur le bouton Quit pour quitter l'outil Zebra.
