Introduction

L'atelier de rendu POV-Ray est un Atelier externe conçu pour faciliter le rendu, mais il offre également des options pour les utilisateurs avancés. L'atelier utilise le moteur de rendu POV-Ray.

Fonctions

Appliquer des textures

Il existe plus de 100 textures prédéfinies que vous pouvez appliquer, mais vous pouvez également définir vos propres textures.

Vignettes et aperçu en temps réel

Pour voir l'impact des options de texture sélectionnées, vous pouvez consulter la vignette pré-rendu ou utiliser l'aperçu en direct pour effectuer le rendu de la texture.

Éclairage

Avec les trois types d'éclairage : éclairage de zone, éclairage ponctuel et éclairage ponctuel, et leurs différentes options, vous pouvez créer un éclairage avancé.

Éclairage indirect (GI)

L'atelier a la possibilité d'activer l'éclairage indirect pour créer des images plus réalistes.

Environnements HDRI

Avec la prise en charge des environnements HDRI, les environnements remarquables sont simples à utiliser.

Fichier inc de l'utilisateur

Les utilisateurs expérimentés qui souhaitent accéder à toutes les options du moteur de rendu POV-Ray peuvent le faire en créant un fichier spécial. Pour plus de détails, voir la page Power User sur notre Wiki.

Installation

Cet atelier peut être installé et mis à jour à partir du Gestionnaire des extensions. Le moteur de rendu POV-Ray utilisé par l'atelier doit être installé séparément. Voir Installer POV. Pour les utilisateurs de Windows, le programme d'installation peut être téléchargé depuis la POV-Ray Download Page, pour les utilisateurs de Linux, il peut généralement être installé depuis le gestionnaire de paquets. Consultez les instructions détaillées sur le wiki de POV-Ray si vous êtes sur Mac.

Pour terminer l'installation, le chemin d'accès à l'exécutable POV-Ray doit être défini dans les préférences de l'atelier :

  • Windows : C:/Program Files/POV-Ray/v3.7/bin/pvengine64.exe (le dossier v*.* peut changer en fonction de la version de POV-Ray)
  • Linux : /usr/bin/povray
  • MacOS : non disponible. Si vous avez plus d'informations, n'hésitez pas à nous en faire part.

Outils

  • OpenDialog : Ouvre le dialogue où la plupart du travail est fait. Ici, vous pouvez appliquer des textures, ajouter des environnements HDRI, etc. et démarrer le rendu.
  • Point Light : Insère une lumière ponctuelle.
  • Area Light : Insère une lumière de zone.
  • Spot Light : Insère un spot lumineux.
