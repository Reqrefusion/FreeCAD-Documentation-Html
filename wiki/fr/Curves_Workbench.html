L'atelier Curves est un atelier externe basé sur Python avec une collection d'outils pour les courbes et les surfaces NURBS. Cet atelier est développé avec FreeCAD Master et OCC 7.4.

Remarque : certains outils peuvent ne pas fonctionner avec les versions antérieures.

Installation

Télécharger l'atelier externe via le Gestionnaire des extensions. Sélectionnez l'option Outils → Gestionnaire des extensions du menu.

Outils

  • Curves Ligne : crée une ligne paramétrique entre deux sommets.
  • Curves Modifier une B-spline : crée une courbe B-spline à main levée.
  • Curves Extension de courbe : prolonge l'arête sélectionnée.
  • Curves Relier courbes : joint les arêtes sélectionnées dans les courbes BSpline.
  • Curves Séparateur courbe : divise l'arête sélectionnée.
  • Curves Discrétisation : discrétise une arête ou un fil.
  • Curves Interpolation : interpole les points avec une courbe BSpline.
  • Curves Peigne paramétrique : crée un tracé en peigne paramétrique sur les arêtes sélectionnées.
  • Curves Outil Zebra : texture zébrée pour l'inspection de surface.
  • Curves Solide paramétrique : crée un solide paramétrique à partir des faces sélectionnées.
  • Curves Courbes iso : crée des courbes iso à partir d'une face.
  • Curves Esquisse sur une surface : applique une esquisse sur une surface.
  • Curves Plan de support : crée un plan de support pour les esquisses.
  • Curves Coller une image SVG : colle l'image SVG du presse-papiers.
  • Curves Profil de l'enveloppe du tube : crée un objet des profils à partir de l'Enveloppe d'un tube.
  • Curves Enveloppe de tube : crée l'enveloppe de tube à partir d'un balayage.
  • Curves Surface de Gordon : crée une surface qui enveloppe un réseau de courbes.
  • Curves Segmenter une surface : segmente une surface sur des isocurves.
  • Curves Vers la console : déplace les objets vers la console.
  • Curves Ressort de compression : crée un ressort de compression.
  • Curves Ligne réfléchie : crée les lignes de réflexion sur une forme, selon une direction de vue.
  • Curves Multi lissage : objets de profil de lissage constitués de plusieurs faces en parallèle.
