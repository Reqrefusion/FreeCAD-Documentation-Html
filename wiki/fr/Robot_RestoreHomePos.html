Description

Permet au robot sélectionné de se déplacer vers sa position de départ.

Utilisation

  1. Sélectionnez le robot dans la Vue en arborescence ou dans la Vue 3D.
  2. Cliquez sur ou choisissez Robot → Passer à la position de départ dans le menu supérieur.

Remarques

  • La commande permet à un seul robot de définir sa position de base.
  • Si aucun ou plus d'un robot est sélectionné, vous serez invité à ne choisir qu'un seul robot.
  • Si aucune position de base n'a été spécifiée via Robot Position de départ, la position du robot telle qu'elle est insérée dans la scène sera utilisée comme position de départ.
