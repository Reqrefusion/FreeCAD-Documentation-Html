Template:TinkerCAD Navigation

{{{Select_name}}} {{{Zoom_name}}} {{{Rotate_view_name}}} {{{Pan_name}}}
{{{Select_text}}} {{{Zoom_text}}} {{{Rotate_view_text}}} {{{Pan_text}}}

This template creates the table that shows the TinkerCAD navigation style.

The text has to be given in the specific fields. In this way the template can be translated into different languages.

Example: 

{{TinkerCAD Navigation
|Select_name=Select
|Zoom_name=Zoom
|Rotate_view_name=Rotate view
|Pan_name=Pan

|Select_text=Press the left mouse button over an object you want to select.

|Zoom_text=Use the mouse wheel to zoom in and out.

|Rotate_view_text=Press the right mouse button, then move the pointer.

|Pan_text=Hold the middle mouse button, then move the pointer.
}}

Result:

Select Zoom Rotate view Pan
Press the left mouse button over an object you want to select. Use the mouse wheel to zoom in and out. Press the right mouse button, then move the pointer. Hold the middle mouse button, then move the pointer.
