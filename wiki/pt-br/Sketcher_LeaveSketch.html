Description

The Sketcher LeaveSketch tool finishes sketch edit mode and closes the Sketcher Dialog.

Usage

  1. There are several ways to invoke the tool:
    • Press the Close button at the top of the Sketcher Dialog.
    • If the Esc can leave sketch edit mode preference is selected (default): press the Esc key.
    • Press the Leave sketch button.
    • Select the Sketch → Leave sketch option from the menu.
    • Right-click in the 3D view and select the Leave sketch option from the context menu.
    • Right-click the sketch in the Tree view and select the Finish editing option from the context menu.
  2. The Sketcher Dialog is closed.
