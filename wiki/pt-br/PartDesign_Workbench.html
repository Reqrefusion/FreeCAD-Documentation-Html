Introdução

A Bancada de trabalho PartDesign fornece ferramentas avançadas para modelagem de peças sólidas complexas. O seu foco principal é a criação de peças mecânicas que podem ser fabricadas e montadas em um produto acabado. Entretanto, os sólidos criados podem ser usados em geral para qualquer outra finalidade, como projeto arquitetônico, análise de elementos finitos, ou usinagem e impressão 3D.

Enquanto a Bancada de trabalho Part é baseado em uma geometria construtiva sólida(CSG) para construir formas, a Bancada de trabalho PartDesign utiliza uma metodologia paramétrica de edição de características, o que significa que um sólido básico é transformado sequencialmente pela adição de características no topo até que a forma final seja obtida. Veja a página edição de recursos para uma explicação mais completa deste processo, e depois veja Criando uma peça simples com PartDesign para começar a criar sólidos.

See the feature editing page for a more complete explanation of this process, and then see Creating a simple component with PartDesign to get started with creating solids.

Uma discussão mais detalhada da bancada de trabalho Part workbench comparada à bancada de trabalho Part Design pode ser encontrada aqui: Part e Part Design.

Ferramentas

As ferramentas de Projeto da Peça estão todas localizadas no menu Part Design e na barra de ferramentas PartDesign que aparece quando você carrega a bancada de trabalho Part Design.

Ferramentas de estrutura

Essas ferramentas, de fato, não fazem parte da bancada de trabalho PartDesign. Elas pertencem à Base Padrão do sistema. Elas foram desenvolvidas na v0.17 com a intenção de serem úteis para organizar um modelo, e criar montagens; como tal, elas são muito úteis quando se trabalha com corpos criados com esta bancada de trabalho.

  • Peça: adiciona um novo recipiente de peças no documento ativo e o torna ativo.
  • Grupo: adiciona um recipiente de Grupo no documento ativo, o que permite organizar os objetos no vista em árvore.

Ferramentas auxiliares de projeto de peças

  • Criar corpo: cria um objeto Corpo no documento ativo e o torna ativo.
  • Create Sketch:
  • Criar esboço: cria um novo esboço em um rosto ou plano selecionado. Se nenhuma face for selecionada enquanto esta ferramenta é executada, o usuário é solicitado a selecionar um plano do painel Tarefas. A interface então muda para o Bancada de trabalho Sketcher em modo de edição de esboço.
  • Aplique um esboço de um lado: mapeia um esboço para um plano previamente selecionado ou para uma face do corpo ativo.
  • Editar esboço: edite o Esboço selecionado.
  • Validate sketch: verifies the tolerance of different points and adjusts them.
  • Check geometry: Checks the geometry of selected objects for errors.
  • Create a shape binder: creates a shape binder referencing geometry from a single parent object.
  • Criar um aglutinante em forma de subobjeto(s): cria um aglutinante de forma referenciando a geometria de um ou mais objetos pai.
  • Criar um clone: cria um clone do corpo selecionado.
  • Create a datum:
  • Criar um plano de referência: cria um plano de referência no corpo ativo.
  • Criar uma linha de referência: cria uma linha de referência no corpo ativo.
  • Criar um ponto de referência: cria um ponto de referência no corpo ativo.
  • Criar um sistema de coordenadas local: cria um sistema de coordenadas local anexado à geometria do datum no corpo ativo.

Ferramentas de modelagem de projeto de peças

Ferramentas aditivas

Estas são ferramentas para criar características básicas ou adicionar material a um corpo sólido existente.

  • Protrusão: extruda um sólido de um esboço selecionado.
  • Revolução: cria um sólido ao girar um esboço em torno de um eixo. O esboço deve formar um perfil fechado.
  • Loft aditivo: cria um sólido, fazendo uma transição entre dois ou mais esboços.
  • Aditivo de varredura: cria um sólido ao varrer um ou mais esboços ao longo de um caminho aberto ou fechado.
  • Hélice aditiva: cria um sólido ao varrer um esboço ao longo de uma hélice.
  • Create an additive primitive:
  • Cubo aditivo: cria um cubo aditivo.
  • Cilindro aditivo: cria um cilindro aditivo.
  • Esfera aditiva: cria uma esfera aditiva.
  • Cone aditivo: cria um cone aditivo.
  • Elipsóide aditiva: cria uma elipsóide aditiva.
  • Toro aditivo: cria um toro aditivo.
  • Prisma aditivo: cria um prisma aditivo.
  • Cunha aditiva: cria uma cunha aditiva.

Ferramentas subtrativas

Estas são ferramentas para subtrair material de um corpo existente.

  • Cavidade: cria uma cavidade a partir do esboço selecionado.
  • Perfuração: cria uma função de perfuração a partir do esboço selecionado. O esboço deve conter um ou mais círculos.
  • Sulco: cria uma ranhura ao girar um esboço em torno de um eixo.
  • Loft subtrativo: cria um sólido, fazendo uma transição entre pelo menos dois esboços e depois o subtrai do corpo ativo.
  • Escaneamento subtrativo: cria uma forma sólida ao varrer um ou mais esboços ao longo de um caminho aberto ou fechado e o subtrai do corpo ativo.
  • Hélice subtrativa: cria uma forma sólida ao varrer um esboço ao longo de uma hélice e o subtrai do corpo ativo. introduced in version 0.19
  • Create a subtractive primitive:
  • Cubo subtrativa: adiciona um cubo subtrativa ao corpo ativo.
  • Cilindro subtrativo: adiciona um cilindro subtrativo ao corpo ativo.
  • Esfera subtrativa: adiciona uma esfera subtrativa ao corpo ativo.
  • Cone subtrativo: adiciona um cone subtrativo ao corpo ativo.
  • Elipsóide subtrativa: adiciona uma elipsóide subtrativa ao corpo ativo.
  • Toro subtrativo: adiciona um toro subtrativo ao corpo ativo.
  • Prisma subtrativo: adiciona um prisma subtrativo ao corpo ativo.
  • ‎Cunha subtrativa: adiciona uma cunha subtrativa ao corpo ativo.

Booleano

  • Operação booleana: importa um ou mais Corpos ou Clones PartDesign para o corpo ativo e aplica uma operação booleana.

Ferramentas de embelezamento

Estas ferramentas aplicam um tratamento às bordas ou faces selecionadas.

  • Filete: filetes (redondos) bordas do corpo ativo.
  • Chanfro: chanfraduras das bordas do corpo ativo.
  • Rascunho: aplica um esboço angular a rostos selecionados do corpo ativo.
  • Espessura: cria uma concha grossa a partir do corpo ativo e abre a(s) face(s) selecionada(s).

Ferramentas de transformação

Estas são ferramentas para transformar as características existentes. Elas permitirão que você escolha quais características transformar.

  • Simetria: espelha uma ou mais características em um plano ou face.
  • Padrão Linear: cria um padrão linear baseado em uma ou mais características.
  • Padrão Polar: cria um padrão polar de uma ou mais características.
  • Transformação múltipla: cria um padrão com qualquer combinação das outras transformações.

Extras

Algumas funcionalidades adicionais encontradas no menu Design da peça:

  • Assistente de projeto da roda dentada: cria um perfil de empena que pode ser extrudido. introduced in version 0.19
  • Assistente de projeto de engrenagem envolvente: cria um perfil de engrenagem involutivo que pode ser extrudado.
  • Assistente de projeto do eixo: Gera um eixo a partir de uma tabela de valores e permite analisar forças e momentos. O eixo é feito com um esboço giratório que pode ser editado.

Itens do menu de contexto

  • Suppressed: checkbox to disable a specific feature without deleting it. introduced in version 1.0
  • Designar como uma função resultante: redefine a ponta, que é a característica exposta fora do Corpo.
  • Mover objeto para outro corpo: move o esboço selecionado, a geometria do datum ou a característica para outro Corpo.
  • Mover objeto após outro objeto: permite reordenar a árvore do corpo movendo o esboço, a geometria dos dados ou a característica selecionada para outra posição na lista de características.

Itens compartilhados com a bancada de trabalho da peça

  • Aparência: determina a aparência de toda a peça (transparência de cores, etc.).
  • Cores definidas: atribui cores aos rostos das peças.

Ferramentas obsoletas

  • Migrate: migrates files from FreeCAD versions below 0.17 to version 0.17. This tool is not available in version 1.0 and above.
