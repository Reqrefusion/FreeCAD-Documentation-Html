Introdução

A bancada de trabalho Web, na verdade, não é uma bancada de modelagem, porém provê uma janela de navegador dentro do FreeCAD.

A partir desse navegador você é capaz de navegar na web, clicar em links, ler documentação online e entrar em websites (cookies são suportados). A bancada Web também é base para ferramentas relacionadas à web no FreeCAD. A janela do navegador é baseada em WebKit e fornece botões de navegação padrão tais como parar, recarregar, próxima página e página anterior, assim também como ampliar e reduzir.

Tools

The Web Workbench offers the following tools:

  • Set URL: Changes the URL of the active Web tab.
  • Open website...: Opens the FreeCAD website.
  • Previous page: Goes to the previous web page.
  • Next page: Goes to the next web page.
  • Refresh web page: Updates the current web page.
  • Stop loading: Stops loading the current web page.
  • Zoom in: Zooms in on the current web page.
  • Zoom out: Zooms out on the current web page.
