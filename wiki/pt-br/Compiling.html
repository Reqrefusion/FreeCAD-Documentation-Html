Compilar no Windows Compilar no Linux Compilar no MacOS Compilar no Docker container

Para compilar em BSD, e outros sistemas Unix, você pode seguir as instruções gerais em Compilar no Linux, mas usando as dependências apropriadas nesses sistemas.

Também é possível compilar o FreeCAD no Windows, seguindo as instruções em Compilar no Windows ou usando MSYS2/MinGW veja Compilar no MinGW.

Para compilar o FreeCAD sem sua interface gráfica de usuário, use a imagem FreeCAD Docker modo CLI.

Para compilar a documentação da programação, ver Documentação do código-fonte.

Para aprender a usar o sistema de controle de revisão e contribuir com o código, veja Gerenciamento de código-fonte.
