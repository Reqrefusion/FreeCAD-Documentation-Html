The FreeGrid Workbench is an external workbench for generating FreeGrid storage system components. In which are included grids, boxes and bit cartridge holders.

Example of a FreeGrid 6x4 grid containing 3 boxes and one bit cartridge holder.

Installation

  1. Install the FreeGrid Workbench via the Addon Manager. For manual installation, see Installing more workbenches.
  2. Restart FreeCAD.

Tools

FreeGrid object generation

  • Storage box: Creates a storage box.
  • Bit cartridge holder: Creates a bit cartridge holder.
  • Storage grid: Creates a storage grid.
  • Sketch: Generates an inner box profile sketch.

Utilities

  • PreferencesPage: Opens the FreeGrid preferences page.
  • About: Opens the FreeGrid "About" page.
