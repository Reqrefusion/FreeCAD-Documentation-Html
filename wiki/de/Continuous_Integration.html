Kontinuierliche Integration

Aktuell stößt das FreeCAD-Repository auf GitHub einen Build auf mehreren CI-Systemen an. Durch diese sind praktisch alle wichtigen Betriebssysteme (Linux, macOS und Windows) abgedeckt. CIs können auch für Erprobung verwendet werden.

Tipps

  • Durch Hinzufügen von [skip ci] oder [ci skip] zu einem git-Commit wird ein CI-Build abgebrochen.

Relevante Verweise

  • Forum-Post der neue CIs ankündigt: New CI is yours :)
