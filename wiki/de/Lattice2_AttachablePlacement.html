Beschreibung

The Lattice2 AttachablePlacement command assigns a placement or handle to an object so it can later be referred to by a location.

Anwendung

  1. Switch to the Lattice2 external workbench
  2. Invoke the Attached Placement command several ways:
    • Press the icon in the Lattice2 toolbar
    • Use the Lattice2 → Attached Placement → Attached Placement from the Lattice2 dropdown
    • Use the {{{1}}} {{{1}}} keyboard shortcut
  3. Choose the options from the dialog menu

Eigenschaften

Hinweise

  • Placement must be made inside the body.
