Einleitung

Der Arbeitsbereich Reinforcement ist ein externer Arbeitsbereich, der Werkzeuge für die Erstellung und Detaillierung von Armierungen (Bewehrungen) bereit stellt. Dieser Arbeitsbereich beinhaltet eine Benutzerschnittstelle und Voreinstellungen zum Erstellen üblicher Armierungen. Er enthält auch Werkzeuge zum Erstellen von Bewehrungsstücklisten, Zuschnittlisten für Bewehrungen, Biegeplänen für Bewehrungen und zum Zeichnen sowie Bemaßen von Bewehrungen.

  • Geradlinige Bewehrung

  • U-förmige Bewehrung

  • L-förmige Bewehrung

  • Bewehrungsbügel

  • Abgesetzte Bewehrung

  • Wendelbewehrung

  • Rechteckige Einzelbügel-Stützenarmierung (Bewehrungskorb)

  • Rechteckige Zwei-Bügel-Stützenarmierung mit sechs Bewehrungsstäben (Bewehrungskorb)

  • Ringförmige Stützenarmierung (Bewehrungskorb)

Installieren

Der Arbeitsbereich Reinforcement ist standardmäßig nicht im FreeCAD-Paket enthalten, kann aber einfach mit dem Addon-Manager installiert werden: Werkzeuge → Addon-Manager → Reinforcement. Der Code des Arbeitsbereichs Reinforcement wird auf github bereitgestellt und weiterentwickelt; er kann auch manuell installiert werden, indem er in FreeCADs Verzeichnis MOD kopiert wird.

Werkzeuge

Armierung erstellen

  • Bewehrung: Erstellt unter Verwendung einer Skizze ein individuell angepasstes Bewehrungselement in einem ausgewählten Strukturelement.
  • Gerade Bewehrung: Erstellt einen geraden Bewehrungsstab in einem ausgewählten Strukturelement.
  • U-förmige Bewehrung: Erstellt einen U-förmigen Bewehrungsstab in einem ausgewählten Strukturelement.
  • L-förmige Bewehrung: Erstellt einen L-förmigen Bewehrungsstab in einem ausgewählten Strukturelement
  • Bewehrungsbügel: Erstellt einen Bewehrungsbügel in einem ausgewählten Strukturelement.
  • Abgesetzte Bewehrung: Erstellt einen Bewehrungsstab mit Absatz in einem ausgewählten Strukturelement.
  • Wendelbewehrung: Erstellt eine Wendelbewehrung in einem ausgewählten Strukturelement.
  • Stützenbewehrung: Erstellt Armierungen (Bewehrungsstäbe und Bewehrungsbügel) in einer ausgewählten Stütze.
  • Balkenbewehrung: Erstellt Armierungen (Bewehrungsstäbe und Bewehrungsbügel) in einem ausgewählten Balken.
  • Plattenbewehrung: Erstellt Bewehrungsstäbe in einer ausgewählten Platte.
  • Fundamentbewehrung: Erstellt Armierungen (Bewehrungsstäbe und Bewehrungsbügel) in einem ausgewählten Fundament.

Armierung detaillieren

  • Armierung Stückliste: Erstellt eine Stückliste der Bewehrungselemente.
  • Armierung Zuschnittliste: Erstellt eine Form- und Zuschnittliste der Bewehrungselemente.
  • Armierung Biegeplan: Erstellt einen Biegeplan der Bewehrungselemente.
  • Armierung ZeichnungBemaßen: Erstellt Zeichnungen mit Bemaßung von den Bewehrungselementen.
