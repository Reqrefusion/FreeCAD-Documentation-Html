Einleitung

Der Arbeitsbereich Stemfie ist ein externer Arbeitsbereich, der zum Ziel hat, eine parametrische Menge von Bauteilen für das Stemfie-Projekt zu implementieren.

Installieren

Der Arbeitsbereich Stemfie ist nicht mit dem Standard FreeCAD-Paket gebündelt, kann aber einfach über den Addon-Manager installiert werden. Dieser wird mit Werkzeuge → Addon-Manager aufgerufen. Der Code des Arbeitsbereichs Stemfie wird auf github bereitgestellt und entwickelt und kann auch manuell installiert werden, indem er in das FreeCAD-Verzeichnis MOD kopiert wird.
