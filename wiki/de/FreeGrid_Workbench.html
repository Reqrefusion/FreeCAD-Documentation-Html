Der Arbeitsbereich FreeGrid is an externer Arbeitsbereich zum Erstellen von FreeGrid storage System-Komponenten, die Raster, Kästen und Werkzeug-Bit-Magazine einschließen.

Beispiel eines 6x4 FreeGrid-Rasters, das 3 Kästen und ein Werkzeug-Bit-Magazin enthält.

Installation

Der Arbeitsbereich FreeGrid kann mit dem Addon-Manager installiert werden. Zur manuellen Installation siehe Weitere Arbeitsbereiche installieren.

Werkzeuge

FreeGrid object generation

  • Storage box: Creates a storage box.
  • Bit cartridge holder: Creates a bit cartridge holder.
  • Storage grid: Creates a storage grid.
  • Sketch: Generates an inner box profile sketch.

Utilities

  • PreferencesPage: Opens the FreeGrid preferences page.
  • About: Opens the FreeGrid "About" page.
