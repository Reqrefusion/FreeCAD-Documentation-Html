Einleitung

Der Arbeitsbereich Mesh (Polygonnetz) behandelt Dreiecksnetze. Netze sind eine besondere Art von 3D-Objekten, die aus dreieckigen Flächen bestehen, die durch ihre Knoten und Kanten miteinander verbunden sind.

Viele 3D-Anwendungen, wie Sketchup, blender, Maya und 3d studio max, verwenden Polygonnetze als wichtigsten Typ von 3D-Objekten. Da Polygonnetze sehr einfache Objekte sind, die nur aus Knoten (Punkten), Kanten und dreieckigen Flächen bestehen, sind sie sehr einfach zu erstellen, zu bearbeiten, zu unterteilen und zu dehnen und können leicht und ohne Verlust an Details von einer Anwendung zur anderen übertragen werden. Außerdem können 3D-Anwendungen in der Regel sehr große Mengen davon ohne Probleme verwalten, da sie sehr einfache Daten enthalten. Aus diesen Gründen sind Netze oft das 3D-Objekt der Wahl, für Anwendungen im Umgang mit Filmen, Animationen und Bilderstellung.

Im technischen Bereich stellen Polygonnetze allerdings eine große Begrenzung dar: Sie können gekrümmte Flächen nicht präzise definieren. Deshalb setzt FreeCAD stattdessen auf Brep. Der Arbeitsbereich Mesh bietet einige Befehle zur direkten Änderung von Polygonnetzen, wird aber am häufigsten verwendet, um 3D-Netzdaten zu importieren und in einen Festkörper umzuwandeln, zur Verwendung mit den Arbeitsbereichen Part oder PartDesign.

Werkzeuge

Auf alle Werkzeuge des Arbeitsbereichs Mesh kann über das Menü Netze zugegriffen werden. Fast alle Werkzeuge sind auch in einer der Mesh-Symbolleisten verfügbar.

  • Netz importieren...: Importiert ein Netzobjekt aus einer Datei.
  • Netz exportieren...: Exportiert ein Netzobjekt in eine Datei.
  • Netz aus Form erstellen...: Erstellt Netzobjekte aus Formobjekten.
  • Aufbereiten...: Vernetzt ein Netzobjekt erneut.
  • Analysieren
    • Netz auswerten und reparieren...: Bewertet und repariert ein Netzobjekt.
    • Flächeninfo: Zeigt Informationen über Flächen von Netzobjekten an.
    • Krümmungsinfo: Zeigt Informationen über Flächen von Netzobjekten an: Zeigt die absolute Krümmung von Krümmungsobjekte an ausgewählten Punkten an.
    • Festkörpernetz prüfen: Prüft, ob ein Netzobjekt fest ist.
    • Begrenzungsinfo...: Zeigt die Begrenzungsrahmen Koordinaten eines Netzobjekts an.
  • Krümmungsdiagramm: Erstellt Netzkrümmungsobjekte für Netzobjekte.
  • Normale harmonisieren: Harmonisiert die Normalen von Netzobjekten.
  • Wendet Normalen: Wendet die Normalen von Netzobjekten.
  • Löcher füllen...: Füllt Löcher in Netzobjekten.
  • Loch schließen: Füllt ausgewählte Löcher in Netzobjekten.
  • Dreieck hinzufügen: Fügt Flächen entlang einer Begrenzung eines offenen Netzobjekts hinzu.
  • Komponenten entfernen...: Entfernt Flächen von Netzobjekten.
  • Komponenten von Hand entfernen...: Entfernt Komponenten von Netzobjekten.
  • Netzsegmente erstellen...: Erstellt separate Netzsegmente für bestimmte Oberflächentypen eines Netzobjekts.
  • Netzsegmente aus bestpassenden Flächen erzeugen...: Erstellt separate Netzsegmente für bestimmte Oberflächentypen eines Netzobjekts und kann deren Parameter identifizieren.
  • Glätten...: Glättet Netzobjekte.
  • Dezimierung...: Reduziert die Anzahl der Flächen in Netzobjekten.
  • Skalieren...: Skaliert Netzobjekte.
  • Regelmäßiger Festkörper... Erzeugt ein regelmäßiges parametrisches Volumenkörper Netzobjekt.
  • Boolsche Operation
  • Vereinigung: Erzeugt ein Netzobjekt, das die Vereinigung von zwei Netzobjekten ist.
  • Schnitt: Erstellt ein Netzobjekt, das die Schnittmenge zweier Netzobjekte ist.
  • Differenz: Erstellt ein Netzobjekt, das die Differenz von zwei Netzobjekten ist.
  • Schneiden
    • Netz schneiden: Schneidet ganze Flächen aus Netzobjekten heraus.
    • Netz trimmen: Trimmt Flächen und Teile von Flächen von Netzobjekten.
    • Netz mit Ebene trimmen: Entfernt Flächen und Teile von Flächen, die sich auf der einen Seite einer Ebene befinden, aus einem Netzobjekt.
    • Schnitt aus Netz und Ebene erstellen: Erzeugt eine ebene Schnittlinie über ein Netzobjekt.
    • Querschnitte...: Erstellt mehrere parallele ebene Schnittlinien über Netzobjekte.
  • Zusammenführen: Erstellt ein Netzobjekt, indem die Netze von zwei oder mehr Netzobjekten kombiniert werden.
  • Komponenten trennen: Zerlegt ein Netzobjekt in seine Bestandteile.
  • Netz abwickeln: Erstellt eine ebene Darstellung eines Netzobjekts.
  • Fläche abwickeln: Erstellt eine ebene Darstellung einer Fläche eines Formobjekts.

Einstellungen

Es gibt einige Exporteinstellungen im Zusammenhang mit Netzformaten, aber diese werden von Befehlen, die zu diesem Arbeitsbereich gehören, nicht verwendet. Sie werden von dem Befehl Std Export verwendet.

Einstellungen des Arbeitsbereichs Mesh können in den folgenden Gruppen des Voreinstellungseditors gefunden werden:

  • Anzeige: Auf der Seite Netzansicht können verschiedene Einstellungen vorgenommen werden.
  • OpenSCAD: Die Befehle Mesh Vereinigung, Mesh Schnitt und Mesh Differenz erfordern OpenSCAD und verwenden die Einstellung OpenSCAD executable, um die ausführbare Datei zu finden.
