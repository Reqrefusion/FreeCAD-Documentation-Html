Beschreibung

Der Befehl Extrudierter Ausschnitt erstellt aus einer Skizze einen extrudierten Ausschnitt.

Der Unterschied zwischen einem 'normalen' Ausschnitt und einem extrudierten Ausschnitt ist, dass bei letzterem die Kanten senkrecht zur ausgewählten Fläche des Blechobjekts ausgeführt werden. Die Auswirkung des Befehls ist nur sichtbar, wenn die Skizze nicht parallel zur Fläche verläuft.

Extrudierter Ausschnitt auf Basis einer kreisförmigen Skizze

Anwendung

  1. Select a planar face and a sketch with a closed contour (in any order).
  2. There are several ways to invoke the command:
    • Press the Extruded Cutout button.
    • Select the SheetMetal → Extruded Cutout option from the menu.
    • Use the keyboard shortcut: E then C.

Eigenschaften

Siehe auch: Eigenschafteneditor.

A SheetMetal ExtrudedCutout object is derived from a Part Feature object or, if it is inside a PartDesign Body, from a PartDesign Feature object, and inherits all its properties. It also has the following additional properties:

Data

Extruded Cutout

  • DatenCut Side (Enumeration): Default is Inside. Side of the cut.
  • DatenCut Type (Enumeration): Default is Through everything both sides. Cut type.
  • Daten (hidden)Extrusion Length1 (Length): Default is 500.0 mm. Length of the extrusion direction 1.
  • Daten (hidden)Extrusion Length2 (Length): Default is 500.0 mm. Length of the extrusion direction 2.
  • DatenSelected Face (LinkSub): The selected object and face.
  • DatenSketch (Link): The sketch for the cut.

Extruded Cutout Improvements

  • DatenImprove Cut (Bool): Default is False. Improve cut geometry if it enters the cutting zone. Only select True if the cut needs fix, because it can be slow.
  • Daten (hidden)Improve Level (IntegerConstraint): Default is 4. Level of cut improvement quality. More than 10 can take a very long time.
  • DatenRefine (Bool): Default is False. Refine the geometry.
