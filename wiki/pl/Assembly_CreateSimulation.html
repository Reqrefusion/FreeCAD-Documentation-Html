Opis

Narzędzie Utwórz symulację tworzy symulację bieżącego złożenia.

Użycie

  1. Jest kilka sposobów na wywołanie tej komendy:
    • Wciśnij przycisk Utwórz symulację.
    • Wybierz opcję Złożenie → Utwórz symulację z menu.
    • Użyj skrótu klawiaturowego: S
  2. ...

Notes

  • The unnamed dialog helps to define an actuator:
    1. Select the desired joint from the Joints list.
    2. If there is more than one transformation available with the selected joint, select one from the Motion Type list.
    3. Edit the Formula that describes the relation between the time running and the driven angle/distance.
    4. Optionally press the Help button to see examples of Formula descriptions.
  • In the Formula time seems to represent 1 rad per second, and with 360° = 6,283 rad for a full rotation we can do either:
    • Enter 6.283 * time in the Formula field and set a duration (End value - Start value) of one second under Simulation settings in the Task panel.
    • Enter 1 * time in the Formula field and set a duration (End value - Start value) of 6.283 seconds under Simulation settings in the Task panel.

Properties

See also: Property editor.

Simulations

A Simulations container is an Assembly::SimulationGroup object which is derived from an App::DocumentObjectGroup object and inherits all its properties. It has no additional properties.

Simulation

A Simulation object is derived from an App FeaturePython object and inherits all its properties. It also has the following additional properties:

Data

Base

  • DANEGroup (LinkList): The Motion objects of the object.
  • DANE (hidden)_ Group Touched (Bool):

Simulation

  • DANEa Time Start (Time): Default is 0.0 s. Simulation start time.
  • DANEb Time End (Time): Default is 4.0 s. Simulation end time.
  • DANEc Time Step Output (Time): Default is 0.01 s. Simulation time step for output.
  • DANEf Global Error Tolerance (Float): Default is 1e-06. Integration global error tolerance.
  • DANEj Frames Per Second (Integer): Default is 30. Frames Per Second.

View

Space

  • WIDOKDecimals (Integer): Default is 9. The number of decimals to use for calculated texts.

Motion

A Motion object is derived from an App FeaturePython object and inherits all its properties. It also has the following additional properties:

Data

Motion

  • DANEFormula (String): The formula of the motion. For example 1.0*time.
  • DANEJoint (XLinkSubHidden): The joint that is moved by the motion.
  • DANEMotion Type (Enumeration): The type of the motion. The options are: Angular and Linear.
