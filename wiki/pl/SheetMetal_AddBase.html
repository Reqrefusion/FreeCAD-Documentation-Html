Opis

Utwórz element bazowy tworzy obiekt bazowy środowiska pracy Arkusz Blachy ze szkicu profilu.

Użycie

Profil

  1. Wybierz szkic otwartego konturu.
  2. Istnieje kilka sposobów na wywołanie tego polecenia:
    • Wciśnij przycisk Dodaj element bazowy na pasku narzędzi.
    • Wybierz opcję Arkusz Blachy → Dodaj element bazowy z menu.
    • Kliknij prawym przyciskiem myszy w widoku drzewa lub widoku 3D i wybierz opcję Arkusz Blachy → Dodaj element bazowy z menu kontekstowego.
    • Użyj skrótu klawiszowego: C a następnie B.
  3. Utworzony zostanie obiekt BaseBend, narożniki wzdłuż konturu będą automatycznie przekształcone w zagięcia cylindryczne.
  4. Dostosuj parametry profilu w Edytorze właściwości:
    • DANEDługość do długości wyciągnięcia profilu,
    • DANEGrubość do grubości ściany profilu,
    • DANEPromień do wewnętrznego promienia automatycznie dodanych zagięć.

Płyta

  1. Wybierz szkic zamkniętego konturu. sketch.
  2. Aktywuj polecenie jak opisano wyżej.
  3. Dostosuj parametry płyty w Edytorze właściwości:
    • DANEGrubość dla grubości płyty.
(DANEDługość oraz DANEPromień są nieużywane dla płyt.)

Task panel

A task panel was introduced in version 0.6.01

Double-click an existing BaseBend object in the Tree view to re-open the task panel and edit the parameters.

  • Sketch: Links a different Sketch to the Bend Sketch property.
  • Bend Plane: Sets the Bend Side property.
  • Radius: Sets the Radius property.
  • Thickness: Sets the Thickness property.
  • Length: Sets the Length property.
  • Symmetric: Toggles the Mid Plane property.
  • Reverse Direction: Toggles the Reverse property.

The last entered values for Radius and Thickness will be saved as default values for the following new BaseBend object.

Właściwości

Zapoznaj się również z informacjami na stronie: Edytor właściwości.

Obiekt bazowy wygięcia środowiska Arkusz Blachy wywodzi się z obiektu Część: Cecha lub, jeśli jest w obrębie Zawartości środowiska Projekt Części, z obiektu Cechy tego środowiska i dziedziczy wszystkie jego właściwości. Posiada on również następujące dodatkowe właściwości:

Dane

Parametry

  • DANEStrona wygięcia (Enumeration): "Typ wpustu", definiuje do której strony krzywej profilu ma zastosowanie grubość. Na zewnątrz (domyślnie), Do wewnątrz, Pośrodku. (nieużywane dla płyt)
  • DANESzkic wygięcia (Link): "Obiekt szkicu ściany". Link do profilu / szkicu konturu.
  • DANEPłaszczyzna środkowa (Bool): "Wyciągnięcie symetrycznie do płaszczyzny". Długość profilu lub grubość płyty dochodzi do jednej strony płaszczyzny szkicu jeśli FAŁSZ (domyślne) lub symetrycznie do obu stron jeśli PRAWDA.
  • DANEOdwrócony (Bool): Odwraca kierunek wyciągnięcia profilu lub grubości płyty. Domyślnie: FAŁSZ.
  • DANEDługość (Length): Długość wyciągnięcia profilu. Domyślnie: 100,00 mm. (nieużywane dla płyt)
  • DANEPromień (Length): Wewnętrzny promień automatycznie dodawanych zagięć. Domyślnie: 1,00 mm. (nieużywane dla płyt)
  • DANEGrubość (Length): Grubość ściany profilu lub płyty. Domyślnie: 1,00 mm.
