Description

The tool SimpleCopy creates a non-parametric copy of a given path.

Usage

  1. Select one CAM object
  2. Press the Simple Copy button

Options

  • The simple copy is not bound anymore to the path that was used to make the copy, and is not parametric. Therefore, when edited with the CAM Inspect tool, any change made to its G-code will be retained.
